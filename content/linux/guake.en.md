> More information: <http://guake-project.org/>.

- Toggle Guake visibility:

```bash
F12
```

- Toggle fullscreen mode:

```bash
F11
```

- Open a new tab:

```bash
Ctrl+Shift+T
```

- Close the terminal:

```bash
Super+X
```

- Go to the previous tab:

```bash
Ctrl+PageUp
```

- Search the selected text in the browser:

```bash
Shift+Ctrl+L
```
