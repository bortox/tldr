> Mais informações: <https://manned.org/free>.

- Mostrar memória do sistema:

```bash
free
```

- Mostrar memória em Bytes/KB/MB/GB:

```bash
free -b|k|m|g
```

- Mostrar memória em unidades legíveis por seres humanos:

```bash
free -h
```

- Atualizar a saída padrão a cada 2 segundos:

```bash
free -s 2
```
