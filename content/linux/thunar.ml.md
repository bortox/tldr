> കൂടുതൽ വിവരങ്ങൾ: <https://docs.xfce.org/xfce/thunar/start>.

- നിലവിലുള്ള ഡയറക്ടറിയിൽ പുതിയ തുണാർ ജാലകം തുറക്കുവാൻ:

```bash
thunar
```

- തൂണാർ ഏതു പതിപ്പാണെന്നറിയുവാൻ:

```bash
thunar --version
```

- തുറന്നിട്ടുള്ള എല്ലാ തുണാർ ജാലകങ്ങളും അവസാനിപ്പിക്കുവാൻ:

```bash
thunar --quit
```
