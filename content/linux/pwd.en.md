> More information: <https://www.gnu.org/software/coreutils/pwd>.

- Print the current directory:

```bash
pwd
```

- Print the current directory, and resolve all symlinks (i.e. show the "physical" path):

```bash
pwd --physical
```

- Print the current logical directory:

```bash
pwd --logical
```
