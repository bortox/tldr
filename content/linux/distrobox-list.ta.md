> மேலும் விவரத்திற்கு: <https://distrobox.privatedns.org>.

- அனைத்து டிஸ்ட்ரோபாக்ஸ் கொள்கலன்களையும் பட்டியலிடுங்கள்:

```bash
distrobox-list
```

- அனைத்து டிஸ்ட்ரோபாக்ஸ் கொள்கலன்களையும் வர்போஸ் தகவலுடன் பட்டியலிடுங்கள்:

```bash
distrobox-list -v
```
