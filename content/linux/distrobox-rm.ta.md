> மேலும் விவரத்திற்கு: <https://distrobox.privatedns.org>.

- டிஸ்ட்ரோபாக்ஸை அகற்றவும்:

```bash
distrobox-rm கொள்கலன்_பெயர்
```

- ஒரு டிஸ்ட்ரோபாக்ஸை வலுக்கட்டாயமாக அகற்றவும்:

```bash
distrobox-rm கொள்கலன்_பெயர் --force
```
