> കൂടുതൽ വിവരങ്ങൾ: <https://networkmanager.dev/docs/api/latest/nmcli.html>.

- nmcli പതിപ്പ് ഏതാണെന്ന് അറിയാൻ:

```bash
nmcli --version
```

- പൊതുവെയുള്ള സഹായ വിവരം കാണാൻ:

```bash
nmcli --help
```

- ഒരു പ്രതേക നിർദേശത്തിന്റെ സഹായ വിവരം കാണാൻ:

```bash
nmcli നിർദേശം --help
```

- ഒരു `nmcli` നിർദേശം നിർവഹിക്കാൻ:

```bash
nmcli നിർദേശം
```
