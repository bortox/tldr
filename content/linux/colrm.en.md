> More information: <https://manned.org/colrm>.

- Remove first column of stdin:

```bash
colrm 1 1
```

- Remove from 3rd column till the end of each line:

```bash
colrm 3
```

- Remove from the 3rd column till the 5th column of each line:

```bash
colrm 3 5
```
