> 更多信息：<https://en.opensuse.org/SDB:Zypper_manual>.

- 同步可用的软件包和版本列表：

```bash
zypper refresh
```

- 安装一个新的软件包：

```bash
zypper install 软件包
```

- 移除一个软件包：

```bash
zypper remove 软件包
```

- 将已安装的软件包升级到最新的可用版本：

```bash
zypper update
```

- 通过关键字搜索软件包：

```bash
zypper search 关键字
```
