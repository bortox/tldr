> Más información: <http://etallen.com/cpuid.html>.

- Muestra información de todas los CPUs:

```bash
cpuid
```

- Muestra información sólo para la CPU actual:

```bash
cpuid -1
```

- Muestra la información hexadecimal en bruto sin decodificar:

```bash
cpuid -r
```
