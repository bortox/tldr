> 更多信息：<https://manpages.debian.org/latest/apache2/a2dissite.8.en.html>.

- 禁用虚拟主机：

```bash
sudo a2dissite 虚拟主机名
```

- 不显示信息性消息：

```bash
sudo a2dissite --quiet 虚拟主机名
```
