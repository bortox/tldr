> Més informació: <https://github.com/aristocratos/bpytop>.

- Inicia bpytop:

```bash
bpytop
```

- Inicia en mode minimalista sense recuadres de memòria y xarxes:

```bash
bpytop -m
```

- Mostra la versió:

```bash
bpytop -v
```

- Canvia a mode minimalista:

```bash
m
```

- Busca processos o programes executant-se:

```bash
f
```

- Canvia ajustaments:

```bash
M
```
