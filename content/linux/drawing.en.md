> More information: <https://maoschanz.github.io/drawing/>.

- Start Drawing:

```bash
drawing
```

- Open specific files:

```bash
drawing path/to/image1 path/to/image2 ...
```

- Open specific files in a new window:

```bash
drawing --new-window path/to/image1 path/to/image2 ...
```
