> 更多信息：<https://manned.org/cal>.

- 显示当前月份的日历：

```bash
cal
```

- 显示上个月、当前月和下个月：

```bash
cal -3
```

- 使用星期一作为一周的第一天：

```bash
cal --monday
```

- 显示特定年份的日历（4位数字）：

```bash
cal 年份
```

- 显示特定月份和年份的日历：

```bash
cal 月份 年份
```
