> Replaces CR with LF.

> More information: <https://waterlan.home.xs4all.nl/dos2unix.html>.

- Change the line endings of a file:

```bash
mac2unix filename
```

- Create a copy with Unix-style line endings:

```bash
mac2unix -n filename new_filename
```
