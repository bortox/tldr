> More information: <https://manned.org/toolbox-list.1>.

- List all `toolbox` containers and images:

```bash
toolbox list
```

- List only `toolbox` containers:

```bash
toolbox list --containers
```

- List only `toolbox` images:

```bash
toolbox list --images
```
