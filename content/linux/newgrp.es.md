> Más información: <https://manned.org/newgrp>.

- Cambia el grupo primario de pertenencia del usuario:

```bash
newgrp nombre_grupo
```

- Restablece el grupo primario de pertenencia al grupo por defecto del usuario `/etc/passwd`:

```bash
newgrp
```
