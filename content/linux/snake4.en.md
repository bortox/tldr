> More information: <https://manpages.debian.org/snake4/snake4.6.en.html>.

- Start a snake game:

```bash
snake4
```

- Choose level:

```bash
1|2|3|4|5
```

- Navigate the snake:

```bash
Up|Down|Left|Right arrow key
```

- Pause game:

```bash
Spacebar
```

- Quit game:

```bash
q
```

- Show the high scores:

```bash
snake4 --highscores
```
