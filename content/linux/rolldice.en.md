> More information: <https://manned.org/rolldice>.

- Roll a single 20 sided dice:

```bash
rolldice d20
```

- Roll two six sided dice and drop the lowest roll:

```bash
rolldice 2d6s1
```

- Roll two 20 sided dice and add a modifier value:

```bash
rolldice 2d20+5
```

- Roll a 20 sided dice two times:

```bash
rolldice 2xd20
```
