> 更多信息：<https://github.com/mellowcandle/bitwise>.

- 使用交互模式运行：

```bash
bitwise
```

- 从十进制转换：

```bash
bitwise 12345
```

- 从十六进制转换：

```bash
bitwise 0x563d
```

- 转换 C 风格的计算：

```bash
bitwise "0x123 + 0x20 - 30 / 50"
```
