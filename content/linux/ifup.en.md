> More information: <https://manpages.debian.org/latest/ifupdown/ifup.8.html>.

- Enable interface eth0:

```bash
ifup eth0
```

- Enable all the interfaces defined with "auto" in `/etc/network/interfaces`:

```bash
ifup -a
```
