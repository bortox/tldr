> More information: <https://manned.org/nmon>.

- Start nmon:

```bash
nmon
```

- Save records to file ("-s 300 -c 288" by default):

```bash
nmon -f
```

- Save records to file with a total of 240 measurements, by taking 30 seconds between each measurement:

```bash
nmon -f -s 30 -c 240
```
