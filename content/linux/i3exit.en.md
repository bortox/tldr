> More information: <https://gitlab.manjaro.org/packages/community/i3/i3exit>.

- Log out of i3:

```bash
i3exit logout
```

- Lock i3:

```bash
i3exit lock
```

- Shut down the system:

```bash
i3exit shutdown
```

- Suspend the system:

```bash
i3exit suspend
```

- Switch to the login screen to log in as a different user:

```bash
i3exit switch_user
```

- Hibernate the system:

```bash
i3exit hibernate
```

- Reboot the system:

```bash
i3exit reboot
```
