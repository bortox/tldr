> 另见 `pacman`。

> 更多信息：<https://github.com/Jguer/yay>.

- 从仓库和 AUR 中交互式搜索和安装软件包：

```bash
yay 软件包|搜索词
```

- 同步并更新所有来自仓库和 AUR 的软件包：

```bash
yay
```

- 只同步和更新 AUR 软件包：

```bash
yay -Sua
```

- 从仓库和 AUR 中安装一个新的软件包：

```bash
yay -S 软件包
```

- 从仓库和 AUR 中搜索软件包数据库中的关键词：

```bash
yay -Ss 关键词
```

- 显示已安装软件包和系统健康状况的统计数据：

```bash
yay -Ps
```
