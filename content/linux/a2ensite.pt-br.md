> Mais informações: <https://manpages.debian.org/latest/apache2/a2ensite.8.en.html>.

- Ativa um host virtual:

```bash
sudo a2ensite host_virtual
```

- Não mostra mensagens informativas:

```bash
sudo a2ensite --quiet host_virtual
```
