> Plus d'informations : <https://manpages.debian.org/latest/apache2/a2ensite.8.en.html>.

- Active un hôte virtuel :

```bash
sudo a2ensite hote_virtuel
```

- N'affiche aucun message (mode silencieux) :

```bash
sudo a2ensite --quiet hote_virtuel
```
