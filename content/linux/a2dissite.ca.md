> Més informació: <https://manpages.debian.org/latest/apache2/a2dissite.8.en.html>.

- Desactiva un host virtual:

```bash
sudo a2dissite host_virtual
```

- No mostris missatges informatius:

```bash
sudo a2dissite --quiet host_virtual
```
