> More information: <https://manned.org/viewnior>.

- View an image:

```bash
viewnior path/to/image.ext
```

- View in fullscreen mode:

```bash
viewnior --fullscreen path/to/image.ext
```

- View fullscreen in slideshow mode:

```bash
viewnior --slideshow path/to/image.ext
```
