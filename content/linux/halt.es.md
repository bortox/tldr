> Más información: <https://www.man7.org/linux/man-pages/man8/halt.8.html>.

- Detiene la máquina:

```bash
halt
```

- Apaga la máquina:

```bash
halt --poweroff
```

- Reinicia la máquina:

```bash
halt --reboot
```
