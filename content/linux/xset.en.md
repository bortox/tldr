> More information: <https://manned.org/xset>.

- Disable the screensaver:

```bash
xset s off
```

- Set the screensaver to start after 60 minutes of inactivity:

```bash
xset s 3600 3600
```

- Disable DPMS (Energy Star) features:

```bash
xset -dpms
```

- Enable DPMS (Energy Star) features:

```bash
xset +dpms
```
