> More information: <https://github.com/Hendrikto/backlight_control>.

- Increase/decrease the backlight by a specific percent count:

```bash
backlight_control +|-5
```

- Set the backlight strength to a specific percent count:

```bash
backlight_control 90
```

- Print the help:

```bash
backlight_control
```
