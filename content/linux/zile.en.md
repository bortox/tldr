> More information: <https://www.gnu.org/software/zile/>.

- Start a buffer for temporary notes, which won't be saved:

```bash
zile
```

- Open a file:

```bash
zile path/to/file
```

- Save a file:

```bash
Ctrl + X, Ctrl + S
```

- Quit:

```bash
Ctrl + X, Ctrl + C
```

- Open a file at a specified line number:

```bash
zile +line_number path/to/file
```

- Undo changes:

```bash
Ctrl + X, U
```
