> More information: <https://github.com/mellowcandle/bitwise>.

- Run using interactive mode:

```bash
bitwise
```

- Convert from decimal:

```bash
bitwise 12345
```

- Convert from hexadecimal:

```bash
bitwise 0x563d
```

- Convert a C-style calculation:

```bash
bitwise "0x123 + 0x20 - 30 / 50"
```
