> 更多信息：<https://i3wm.org/docs/userguide.html>.

- 启动 i3（注意在此命令运行前，务必关闭其它先前存在的窗口管理器）：

```bash
i3
```

- 打开新终端窗口：

```bash
Super + Return
```

- 创建新工作区：

```bash
Super + Shift + 数字键
```

- 切换到 {{数字}} 号工作区：

```bash
Super + 数字键
```

- 水平布局打开新窗口：

```bash
Super + h
```

- 垂直布局打开新窗口：

```bash
Super + v
```

- 打开应用（在执行命令后输入应用名称）：

```bash
Super + D
```
