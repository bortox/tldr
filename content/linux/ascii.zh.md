> 更多信息：<http://www.catb.org/~esr/ascii/>.

- 显示一个 ASCII 字符的别名：

```bash
ascii a
```

- 以较短和脚本友好模式显示 ASCII 字符的别名：

```bash
ascii -t a
```

- 显示多个 ASCII 字符的别名：

```bash
ascii -s tldr
```

- 显示十进制 ASCII 字符表：

```bash
ascii -d
```

- 显示十六进制 ASCII 字符表：

```bash
ascii -x
```

- 显示八进制 ASCII 字符表：

```bash
ascii -o
```

- 显示二进制 ASCII 字符表：

```bash
ascii -b
```

- 显示选项总结和整个 ASCII 字符表：

```bash
ascii
```
