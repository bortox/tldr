- Starte ein neues Spiel:

```bash
cuyo
```

- Verschiebe die Teile horizontal:

```bash
A|D oder Pfeiltaste Links|Rechts
```

- Drehe die Teile:

```bash
W|Pfeiltaste nach oben
```

- Lasse die Teile schnell fallen:

```bash
S|Pfeiltaste nach unten
```
