> See also: `lvm`.

> More information: <https://man7.org/linux/man-pages/man8/vgdisplay.8.html>.

- Display information about all volume groups:

```bash
sudo vgdisplay
```

- Display information about volume group vg1:

```bash
sudo vgdisplay vg1
```
