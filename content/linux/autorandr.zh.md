> 更多信息：<https://github.com/phillipberndt/autorandr>.

- 保存当前屏幕布局：

```bash
autorandr -s 配置文件名
```

- 显示保存的配置：

```bash
autorandr
```

- 切换设置：

```bash
autorandr -l 配置文件名
```

- 设置默认设置：

```bash
autorandr -d 配置文件名
```
