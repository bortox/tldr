> More information: <https://manned.org/pi>.

- Display 100 decimal digits of Archimedes' constant Pi:

```bash
pi
```

- Display a specified number of decimal digits of Archimedes' constant Pi:

```bash
pi number
```

- Display help:

```bash
pi --help
```

- Display version:

```bash
pi --version
```

- Display recommended readings:

```bash
pi --bibliography
```
