> More information: <https://manned.org/expect>.

- Execute an expect script from a file:

```bash
expect path/to/file
```

- Execute a specified expect script:

```bash
expect -c "commands"
```

- Enter an interactive REPL (use `exit` or Ctrl + D to exit):

```bash
expect -i
```
