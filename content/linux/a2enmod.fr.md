> Plus d'informations : <https://manpages.debian.org/latest/apache2/a2enmod.8.en.html>.

- Active un module :

```bash
sudo a2enmod module
```

- N'affiche aucun message (mode silencieux) :

```bash
sudo a2enmod --quiet module
```
