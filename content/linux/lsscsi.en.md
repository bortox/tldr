> More information: <https://manned.org/lspci>.

- List all SCSI devices:

```bash
lsscsi
```

- List all SCSI devices with detailed attributes:

```bash
lsscsi -L
```

- List all SCSI devices with human-readable disk capacity:

```bash
lsscsi -s
```
