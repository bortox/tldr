
- Apri un terminale:

```bash
Super + Invio
```

- Crea un nuovo workspace:

```bash
Super + Shift + numero
```

- Salta a workspace {{numero}}:

```bash
Super + numero
```

- Per il prossimo nuovo, dividi orizzontalmente:

```bash
Super + h
```

- Per il prossimo nuovo, dividi verticalmente:

```bash
Super + v
```

- Avvia dmenu, lanciatore di programmi:

```bash
Super + D
```
