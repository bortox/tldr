> 更多信息：<https://flameshot.org>.

- 在 GUI 模式下启动 Flameshot：

```bash
flameshot launcher
```

- 通过单击并拖动来截取屏幕截图：

```bash
flameshot gui
```

- 全屏截图：

```bash
flameshot full
```

- 将保存屏幕快照的路径设置为：

```bash
flameshot full --path path/to/directory
```

- 将屏幕截图延迟 2000 毫秒，然后输出到剪贴板：

```bash
flameshot full --delay 2000 --clipboard
```
