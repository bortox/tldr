> 更多信息：<https://wiki.archlinux.org/index.php/Baloo>.

- 查看帮助：

```bash
balooctl
```

- 显示索引器状态：

```bash
balooctl status
```

- 开启或关闭文件索引器：

```bash
balooctl enable|disable
```

- 清除索引数据库：

```bash
balooctl purge
```

- 挂起文件索引器：

```bash
balooctl suspend
```

- 恢复文件索引器：

```bash
balooctl resume
```

- 显示 Baloo 占用的磁盘空间大小：

```bash
balooctl indexSize
```

- 检查未索引的文件并索引它们：

```bash
balooctl check
```
