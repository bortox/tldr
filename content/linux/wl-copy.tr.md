> Ayrıca bakın: `wl-paste`.

> Daha fazla bilgi: <https://github.com/bugaevc/wl-clipboard>.

- Metni panoya kopyala:

```bash
wl-copy metin
```

- Komutun çıktısını panoya kopyala:

```bash
komut | wl-copy
```

- Yalnızca bir yapıştırma için kopyala ve ardından temizle:

```bash
wl-copy --paste-once
```

- Panoyu temizle:

```bash
wl-copy --clear
```
