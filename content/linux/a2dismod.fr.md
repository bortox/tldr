> Plus d'informations : <https://manpages.debian.org/latest/apache2/a2dismod.8.en.html>.

- Désactive un module :

```bash
sudo a2dismod module
```

- N'affiche aucun message (mode silencieux) :

```bash
sudo a2dismod --quiet module
```
