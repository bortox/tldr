> More information: <https://manned.org/dunstctl>.

- Pause notifications:

```bash
dunstctl set-paused true
```

- Un-pause notifications:

```bash
dunstctl set-paused false
```

- Close all notifications:

```bash
dunstctl close-all
```

- Display help:

```bash
dunstctl --help
```
