> 通过 IMAP 支持大多数现代电子邮件服务。

> 更多信息：<https://manned.org/alpine>.

- 正常打开 alpine：

```bash
alpine
```

- 直接打开写信息界面，并指定电子邮件发送地址：

```bash
alpine 邮箱地址
```

- 退出 alpine：

```bash
'q' 然后 'y'
```
