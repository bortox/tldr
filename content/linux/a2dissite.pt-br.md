> Mais informações: <https://manpages.debian.org/latest/apache2/a2dissite.8.en.html>.

- Desativa um host virtual:

```bash
sudo a2dissite host_virtual
```

- Não mostra mensagens informativas:

```bash
sudo a2dissite --quiet host_virtual
```
