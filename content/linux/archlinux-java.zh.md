> 更多信息：<https://github.com/michaellass/archlinux-java-run>.

- 列出已安装的 Java 环境：

```bash
archlinux-java status
```

- 设置默认的 Java 环境：

```bash
archlinux-java set java_environment
```

- 取消默认的 Java 环境：

```bash
archlinux-java unset
```

- 自动设置默认的 Java 环境：

```bash
archlinux-java fix
```
