> 给定字符串中的每个字符必须单独加下划线。

> 更多信息：<https://manned.org/ul>.

- 在适用的情况下用下划线显示文件的内容：

```bash
ul 文件路径
```

- 用由破折号组成的下划线显示文件的内容 `-`：

```bash
ul -i 文件路径
```
