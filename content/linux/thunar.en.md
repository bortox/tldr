> More information: <https://docs.xfce.org/xfce/thunar/start>.

- Open a new window showing the current directory:

```bash
thunar
```

- Open the bulk rename utility:

```bash
thunar --bulk-rename
```

- Close all open thunar windows:

```bash
thunar --quit
```
