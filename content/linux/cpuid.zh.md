> 更多信息：<http://etallen.com/cpuid.html>.

- 显示所有 CPU 的信息：

```bash
cpuid
```

- 仅显示当前 CPU 的信息：

```bash
cpuid -1
```

- 显示原始十六进制信息，不进行解码：

```bash
cpuid -r
```
