> More information: <https://www.karimmi.de/cuyo/>.

- Start a new game:

```bash
cuyo
```

- Navigate the piece horizontally:

```bash
A|D OR Left|Right arrow key
```

- Turn the piece:

```bash
W|Up arrow key 
```

- Hard drop the piece:

```bash
S|Down arrow key
```
