> Mais informações: <https://manpages.debian.org/latest/apache2/a2dismod.8.en.html>.

- Desativa um módulo:

```bash
sudo a2dismod módulo
```

- Não mostra mensagens informativas:

```bash
sudo a2dismod --quiet módulo
```
