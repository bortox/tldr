> See also `tlp-stat`.

> More information: <https://linrunner.de/tlp/>.

- Apply settings (according to the actual power source):

```bash
sudo tlp start
```

- Apply battery settings (ignoring the actual power source):

```bash
sudo tlp bat
```

- Apply AC settings (ignoring the actual power source):

```bash
sudo tlp ac
```
