> Maggiori informazioni: <https://manpages.debian.org/latest/apache2/a2dismod.8.en.html>.

- Disattiva un modulo:

```bash
sudo a2dismod modulo
```

- Non mostrare messaggi informativi:

```bash
sudo a2dismod --quiet modulo
```
