> Weitere Informationen: <https://manned.org/ldd>.

- Zeige Abhängigkeiten von dynamischen Bibliotheken einer Binärdatei an:

```bash
ldd pfad/zu/binärdatei
```

- Zeige ungenutzte direkte Abhängigkeiten an:

```bash
ldd -u pfad/zu/binärdatei
```
