> Més informació: <https://docs.oracle.com/cd/E19253-01/816-5166/pkgadd-1m/index.html>.

- Instal·la un paquet de software local:

```bash
pkgadd nom_paquet
```

- Actualitza un paquet ja instal·lat a partir de un paquet local:

```bash
pkgadd -u nom_paquet
```
