> 更多信息：<https://github.com/pavanjadhaw/betterlockscreen>.

- 锁定屏幕：

```bash
betterlockscreen --lock
```

- 更换锁屏背景：

```bash
betterlockscreen -u 图片.png
```

- 锁屏并显示一些自定义文本：

```bash
betterlockscreen -l pixel -t "自定义锁屏文本"
```

- 锁屏并自定义显示屏关闭超时秒数：

```bash
betterlockscreen --off 5 -l
```
