> Plus d'informations : <https://manpages.debian.org/latest/ifupdown/ifup.8.html>.

- Active l'interface eth0 :

```bash
ifup eth0
```

- Active l'ensemble des interfaces réseau définies dans le fichier `/etc/network/interfaces` :

```bash
ifup -a
```
