> 更多信息：<https://github.com/aristocratos/bpytop>.

- 运行 bpytop：

```bash
bpytop
```

- 以最小模式启动，无需内存和网络盒：

```bash
bpytop -m
```

- 显示版本：

```bash
bpytop -v
```

- 切换最小模式：

```bash
m
```

- 搜索正在运行的程序或进程：

```bash
f
```

- 改变设置：

```bash
M
```
