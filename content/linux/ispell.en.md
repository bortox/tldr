> More information: <https://www.cs.hmc.edu/~geoff/ispell.html>.

- Start an interactive session:

```bash
ispell
```

- Check for typos in the specified file and interactively apply suggestions:

```bash
ispell path/to/file
```

- Display the version:

```bash
ispell -v
```
