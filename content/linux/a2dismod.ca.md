> Més informació: <https://manpages.debian.org/latest/apache2/a2dismod.8.en.html>.

- Desactiva un mòdul:

```bash
sudo a2dismod mòdul
```

- No mostrius missatges informatius:

```bash
sudo a2dismod --quiet mòdul
```
