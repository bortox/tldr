> More information: <http://www.mike-devlin.com/linux/README-numlockx.htm>.

- Show the current number lock status:

```bash
numlockx status
```

- Turn the number lock on:

```bash
numlockx on
```

- Turn the number lock off:

```bash
numlockx off
```

- Toggle the current state:

```bash
numlockx toggle
```
