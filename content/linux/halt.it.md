> Maggiori informazioni: <https://www.man7.org/linux/man-pages/man8/halt.8.html>.

- Arresta la macchina:

```bash
halt
```

- Spegne la macchina:

```bash
halt --poweroff
```

- Riavvia la macchina:

```bash
halt --reboot
```
