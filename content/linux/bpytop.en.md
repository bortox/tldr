> More information: <https://github.com/aristocratos/bpytop>.

- Start bpytop:

```bash
bpytop
```

- Start in minimal mode without memory and networking boxes:

```bash
bpytop -m
```

- Show version:

```bash
bpytop -v
```

- Toggle minimal mode:

```bash
m
```

- Search for running programs or processes:

```bash
f
```

- Change settings:

```bash
M
```
