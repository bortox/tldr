> Plus d'informations : <https://wiki.termux.com/wiki/Package_Management>.

- Mets à jour tous les paquets déjà installés :

```bash
pkg upgrade
```

- Installe un paquet :

```bash
pkg install paquet
```

- Désinstalle un paquet :

```bash
pkg uninstall paquet
```

- Réinstalle un paquet :

```bash
pkg reinstall paquet
```

- Cherche un paquet :

```bash
pkg search paquet
```
