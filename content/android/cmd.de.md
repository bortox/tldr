- Liste alle laufenden Services auf:

```bash
cmd -l
```

- Rufe einen bestimmten Service auf:

```bash
cmd alarm
```

- Rufe einen bestimmten Service mit Parametern auf:

```bash
cmd vibrator vibrate 300
```
