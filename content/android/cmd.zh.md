> 更多信息：<https://cs.android.com/android/platform/superproject/+/master:frameworks/native/cmds/cmd/>.

- 列出所有正在运行的服务：

```bash
cmd -l
```

- 调用指定服务：

```bash
cmd alarm
```

- 调用服务同时传递参数：

```bash
cmd vibrator vibrate 300
```
