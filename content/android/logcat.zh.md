> 更多信息：<https://developer.android.com/studio/command-line/logcat>.

- 显示系统日志：

```bash
logcat
```

- 将系统日志写入文件：

```bash
logcat -f 文件路径
```

- 显示与正则表达式匹配的行：

```bash
logcat --regex 正则表达式
```
