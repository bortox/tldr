> Mais informações: <https://wiki.termux.com/wiki/Package_Management>.

- Atualiza todos os pacotes instalados:

```bash
pkg upgrade
```

- Instala um pacote:

```bash
pkg install pacote
```

- Desinstala um pacote:

```bash
pkg uninstall pacote
```

- Reinstala um pacote:

```bash
pkg reinstall pacote
```

- Busca por um pacote:

```bash
pkg search pacote
```
