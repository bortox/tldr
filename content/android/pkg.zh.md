> 更多信息：<https://wiki.termux.com/wiki/Package_Management>.

- 升级所有已安装的包：

```bash
pkg upgrade
```

- 安装一个包：

```bash
pkg install 包名
```

- 卸载一个包：

```bash
pkg uninstall 包名
```

- 重新安装一个包：

```bash
pkg reinstall 包名
```

- 搜索一个包：

```bash
pkg search 包名
```
