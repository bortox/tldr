> More information: <https://picard.musicbrainz.org/>.

- Start Picard:

```bash
picard
```

- Open a set of files:

```bash
picard path/to/file1.mp3 path/to/file2.mp3
```

- Display the version of Picard installed:

```bash
picard --long-version
```
