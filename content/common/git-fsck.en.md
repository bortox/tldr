> Does not make any modifications. See `git gc` for cleaning up dangling blobs.

> More information: <https://git-scm.com/docs/git-fsck>.

- Check the current repository:

```bash
git fsck
```

- List all tags found:

```bash
git fsck --tags
```

- List all root nodes found:

```bash
git fsck --root
```
