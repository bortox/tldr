> More information: <https://manned.org/most>.

- Open a file:

```bash
most path/to/file
```

- Open several files:

```bash
most path/to/file1 path/to/file2
```

- Open a file at the first occurrence of "string":

```bash
most file +/string
```

- Move through opened files:

```bash
:O n
```

- Jump to the 100th line:

```bash
100j
```

- Edit current file:

```bash
e
```

- Split the current window in half:

```bash
<CTRL-x> o
```

- Exit:

```bash
Q
```
