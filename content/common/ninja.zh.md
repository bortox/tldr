> 更多信息：<https://ninja-build.org/manual.html>.

- 在当前目录下构建：

```bash
ninja
```

- 在指定的目录中构建一个程序：

```bash
ninja -C 路径
```

- 查看 target（如 `install` 和 `uninstall`）：

```bash
ninja -t targets
```

- 查看帮助：

```bash
ninja -h
```
