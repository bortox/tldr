> More information: <https://secure.phabricator.com/book/phabricator/article/arcanist/>.

- Send the changes to Differential for review:

```bash
arc diff
```

- Show pending revision information:

```bash
arc list
```

- Update Git commit messages after review:

```bash
arc amend
```

- Push Git changes:

```bash
arc land
```
