> More information: <https://jubalh.github.io/nudoku/>.

- Start a sudoku game:

```bash
nudoku
```

- Choose the difficulty of the game:

```bash
nudoku -d easy|normal|hard
```

- Navigate the board:

```bash
h|j|k|l OR Left|Down|Up|Right arrow key
```

- Delete a number:

```bash
Backspace|x
```

- Get a hint:

```bash
H
```

- See the complete solution:

```bash
S
```

- Create a new puzzle:

```bash
N
```

- Quit the game:

```bash
Q
```
