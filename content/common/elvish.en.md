> See also: `nu`.

> More information: <https://elv.sh>.

- Start an interactive shell session:

```bash
elvish
```

- Execute specific [c]ommands:

```bash
elvish -c "echo 'elvish is executed'"
```

- Execute a specific script:

```bash
elvish path/to/script.elv
```
