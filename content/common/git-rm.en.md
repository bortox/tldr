> More information: <https://git-scm.com/docs/git-rm>.

- Remove file from repository index and filesystem:

```bash
git rm file
```

- Remove directory:

```bash
git rm -r directory
```

- Remove file from repository index but keep it untouched locally:

```bash
git rm --cached file
```
