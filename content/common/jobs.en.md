> More information: <https://manned.org/jobs>.

- Show status of all jobs:

```bash
jobs
```

- Show status of a particular job:

```bash
jobs %job_id
```

- Show status and process IDs of all jobs:

```bash
jobs -l
```

- Show process IDs of all jobs:

```bash
jobs -p
```
