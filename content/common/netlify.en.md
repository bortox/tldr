> More information: <https://cli.netlify.com>.

- Log in to the Netlify account:

```bash
netlify login
```

- Deploy the contents of a directory to Netlify:

```bash
netlify deploy
```

- Configure continuous deployment for a new or an existing site:

```bash
netlify init
```

- Start a local dev server:

```bash
netlify dev
```
