> 더 많은 정보: <https://couchdb.apache.org>.

- couchdb 시작하기:

```bash
couchdb
```

- couchdb 대화식 쉘 시작하기:

```bash
couchdb -i
```

- 백그라운드 프로세스로 couchdb 시작하기:

```bash
couchdb -b
```

- 백그라운드 프로세스 종료(참고 : 필요한 경우 다시 생성됨):

```bash
couchdb -k
```

- 백그라운드 프로세스 종료:

```bash
couchdb -d
```
