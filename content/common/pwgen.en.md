> More information: <https://github.com/tytso/pwgen>.

- Generate random password with s[y]mbols:

```bash
pwgen -y length
```

- Generate secure, hard-to-memorize passwords:

```bash
pwgen -s length
```

- Generate password with at least one capital letter in them:

```bash
pwgen -c length
```
