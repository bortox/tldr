> See also `histexpand` for history expansion.

> More information: <https://manned.org/sh>.

- Start an interactive shell session:

```bash
sh
```

- Execute a command and then exit:

```bash
sh -c "command"
```

- Execute a script:

```bash
sh path/to/script.sh
```

- Read and execute commands from stdin:

```bash
sh -s
```
