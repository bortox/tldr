> Incluso in csvkit.

> Maggiori informazioni: <https://csvkit.readthedocs.io/en/latest/scripts/csvclean.html>.

- Correggi un file CSV:

```bash
csvclean file.csv
```

- Elenca le posizioni di errori di sintassi in un file CSV:

```bash
csvclean -n file.csv
```
