> 更多信息：<https://manned.org/more>.

- 打开一个文件：

```bash
more 路径/到/文件
```

- 打开一个文件，从特定行开始显示：

```bash
more +行号 路径/到/文件
```

- 显示帮助：

```bash
more --help
```

- 转到下一页：

```bash
<空格>
```

- 搜索一个字符串（按 `n` 键跳转到下一个匹配）:

```bash
/字符串
```

- 退出：

```bash
q
```

- 显示关于交互式命令的帮助：

```bash
h
```
