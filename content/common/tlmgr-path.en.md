> This command has to be re-run for files added in the future.

> More information: <https://www.tug.org/texlive/tlmgr.html>.

- Add symlinks to TeX Live files:

```bash
sudo tlmgr path add
```

- Remove symlinks to TeX Live files:

```bash
sudo tlmgr path remove
```
