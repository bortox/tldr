> Weitere Informationen: <https://manned.org/exit>.

- Beende die Shell mit dem Exitcode des zuletzt ausgeführten Befehls:

```bash
exit
```

- Beende die Shell mit dem angegebenen Exitcode:

```bash
exit exitcode
```
