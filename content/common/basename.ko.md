> 더 많은 정보: <https://www.gnu.org/software/coreutils/basename>.

- 경로에서 파일 이름만 표시:

```bash
basename path/to/file
```

- 경로에서 접미사가 제거된 파일 이름만 표시:

```bash
basename path/to/file suffix
```
