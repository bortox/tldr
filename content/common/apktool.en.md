> More information: <https://ibotpeaches.github.io/Apktool/>.

- Decode an APK file:

```bash
apktool d file.apk
```

- Build an APK file from a directory:

```bash
apktool b path/to/directory
```

- Install and store frameworks:

```bash
apktool if framework.apk
```
