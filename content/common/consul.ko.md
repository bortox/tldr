> 더 많은 정보: <https://www.consul.io/commands>.

- Consul 버전을 체크합니다:

```bash
consul --version
```

- 일반 도움말을 보여줍니다:

```bash
consul --help
```

- 서브 명령어를 위한 도움말을 보여줍니다:

```bash
consul 하위-명령어 --help
```
