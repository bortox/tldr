> 更多資訊：<https://www.gnu.org/software/coreutils/pwd>.

- 印出目前所在的目錄名稱：

```bash
pwd
```

- 印出目前所在的目錄名稱，並將符號連結轉換為實體路徑：

```bash
pwd -P
```
