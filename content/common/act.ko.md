> 더 많은 정보: <https://github.com/nektos/act>.

- 가능한 작업들 목록:

```bash
act -l
```

- 기본 이벤트 실행:

```bash
act
```

- 특정 이벤트 실행:

```bash
act event_type
```

- 특정 작업 실행:

```bash
act -a action_id
```

- 실제론 작업을 실행하지 않기 (예 : a dry run):

```bash
act -n
```

- 자세한 로그 표시:

```bash
act -v
```
