> Maggiori informazioni: <https://git-scm.com/docs/git-init>.

- Inizializza un nuovo repository locale:

```bash
git init
```

- Inizializza un repository di soli dati, adatto per essere usato come server remoto accessibile via ssh:

```bash
git init --bare
```
