> More information: <https://maestral.app/cli>.

- Start the GUI:

```bash
maestral gui
```

- Print current status of Maestral:

```bash
maestral status
```

- Pause syncing:

```bash
maestral pause
```

- Resume syncing:

```bash
maestral resume
```

- Print sync status of a specific file or folder:

```bash
maestral filestatus path/to/file_or_directory
```
