> 더 많은 정보: <https://docs.microsoft.com/cli/azure>.

- Azure에 로그인:

```bash
az login
```

- Azure 구독 정보 관리:

```bash
az account
```

- 모든 Azure 관리 디스크 나열:

```bash
az disk list
```

- 모든 Azure 가상머신 나열:

```bash
az vm list
```

- Azure Kubernetes Services 관리:

```bash
az aks
```

- Azure 네트워크 리소스 관리:

```bash
az network
```
