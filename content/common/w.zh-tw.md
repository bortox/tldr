> 更多資訊：<https://manned.org/w>.

- 顯示目前的登錄者的訊息：

```bash
w
```

- 顯示特定登錄者的訊息：

```bash
w 使用者名稱
```

- 顯示目前的登錄者的訊息，並隱藏標題：

```bash
w --no-header
```

- 顯示 login、JCPU、PCPU 以外的資訊：

```bash
w --short
```
