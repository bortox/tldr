> Mais informações: <https://manned.org/sh>.

- Inicie uma sessão shell interativa:

```bash
sh
```

- Execute um comando e então saia da shell:

```bash
sh -c "comando"
```

- Execute um script:

```bash
sh caminho/para/script.sh
```

- Leia e execute comandos da entrada stdin (entrada padrão):

```bash
sh -s
```
