> More information: <https://github.com/RazrFalcon/svgcleaner>.

- Optimize an SVG:

```bash
svgcleaner input.svg output.svg
```

- Optimize an SVG multiple times:

```bash
svgcleaner --multipass input.svg output.svg
```
