> Il ne doit pas être utilisé manuellement.

> Plus d'informations : <https://manned.org/auditd>.

- Démarre le démon :

```bash
auditd
```

- Démarre le démon en mode débogage :

```bash
auditd -d
```

- Démarre le démon à la demande depuis launchd :

```bash
auditd -l
```
