> More information: <https://symfony.com>.

- Create a new Symfony project:

```bash
symfony new name
```

- Run a local web server:

```bash
symfony serve
```

- Stop the local web server:

```bash
symfony server:stop
```

- Check for security issues in the project's dependencies:

```bash
symfony security:check
```
