> Maggiori informazioni: <http://elinks.or.cz>.

- Avvia elinks:

```bash
elinks
```

- Termina elinks:

```bash
Ctrl + C
```

- Stampa l'output di una pagina web nella console, colorando il testo con codici di controllo ANSI:

```bash
elinks -dump -dump-color-mode 1 url
```
