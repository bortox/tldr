> 更多信息：<https://github.com/tj/n>.

- 安装给定版本的节点。如果版本已经安装，它将被激活：

```bash
n 版本
```

- 显示已安装的版本并以交互方式激活其中一个版本：

```bash
n
```

- 删除一个版本：

```bash
n rm 版本
```

- 使用给定版本执行文件：

```bash
n use 版本 文件.js
```

- 输出指定版本的二进制：

```bash
n bin 版本
```
