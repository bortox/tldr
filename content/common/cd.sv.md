> Mer information: <https://manned.org/cd>.

- Gå till en given katalog:

```bash
cd sökväg/till/katalog
```

- Gå till hemkatalog för nuvarande användare:

```bash
cd
```

- Gå till överordnad katalog:

```bash
cd ..
```

- Gå tillbaks till den föregående valda katalogen:

```bash
cd -
```
