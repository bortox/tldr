> 更多信息：<https://tldr.sh>.

- 获取一个命令的用例（提示：这就是你怎么得到本信息的）：

```bash
tldr command
```

- 展示 Linux 下 tar 的 tldr 文档：

```bash
tldr -p linux tar
```

- 获取一个 Git 子命令的帮助：

```bash
tldr git checkout
```

- 更新本地页面（如果客户端支持缓存）：

```bash
tldr -u
```
