> Više informacija: <https://tldr.sh>.

- Prikaži tipičnu upotrebu komande (pomoć: ovako ste stigli ovde!):

```bash
tldr komanda
```

- Prikaži tar tldr stranicu za Linux:

```bash
tldr -p linux tar
```

- Prikaži pomoć za Git potkomandu:

```bash
tldr git-checkout
```
