> Weitere Informationen: <https://manned.org/rev>.

- Kehre die Reihenfolge des Textes "Hallo" um:

```bash
echo "Hallo" | rev
```

- Kehre die Reihenfolge einer Datei um:

```bash
rev pfad/zu/datei
```
