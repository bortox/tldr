> More information: <https://gnucash.org>.

- Launch GnuCash and load the previously opened file:

```bash
gnucash
```

- Launch GnuCash and load the specified file:

```bash
gnucash path/to/file.gnucash
```

- Launch GnuCash and load an empty file:

```bash
gnucash --nofile
```
