> 詳しくはこちら: <https://www.gnu.org/software/coreutils/pwd>

- 現在のディレクトリを表示する:

```bash
pwd
```

- 現在のディレクトリを表示し、すべてのシンボリックリンクを解決する（つまり、「物理的な」パスを表示する）:

```bash
pwd -P
```
