> 더 많은 정보: <https://github.com/ptrofimov/clockwork-cli>.

- 현재 프로젝트의 Clockwork 로그들을 모니터링 합니다:

```bash
clockwork-cli
```

- 특정 프로젝트의 Clockwork 로그들을 모니터링 합니다:

```bash
clockwork-cli 경로/디렉토리
```

- 여러 개의 프로젝트의 Clockwork 로그들을 모니터링 합니다:

```bash
clockwork-cli 경로/디렉토리1 경로/디렉토리2 …
```
