> Daha fazla bilgi: <https://git-scm.com/docs/git-reflog>.

- HEAD için referans kaydını göster:

```bash
git reflog
```

- Belirtilen dal için referans kaydını göster:

```bash
git reflog dal_ismi
```

- Referans kaydında sadece son 5 değişimi göster:

```bash
git reflog -n 5
```
