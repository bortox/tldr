> More information: <https://flask.palletsprojects.com/en/1.1.x/cli/>.

- Run a development server:

```bash
flask run
```

- Show the routes for the app:

```bash
flask routes
```

- Run a Python interactive shell in the app's context:

```bash
flask shell
```
