> 더 많은 정보: <https://github.com/chonla/cotton>.

- 특정 기본 URL 사용하기:

```bash
cotton -u 기본_url 파일.md
```

- 인증서 확인 비활성화(비 보안 모드):

```bash
cotton -u 기본_url -i 파일.md
```

- 테스트 실패시 실행 중지:

```bash
cotton -u 기본_url -s 파일.md
```
