> 更多信息：<https://www.gnu.org/software/coreutils/basename>.

- 仅显示文件名：

```bash
basename 路径/到/文件
```

- 显示路径字符最右边表示目录的字符：

```bash
basename 路径/到/目录/
```

- 展示无后缀的文件名称：

```bash
basename 路径/到/文件 后缀
```
