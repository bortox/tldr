> More information: <https://github.com/LuRsT/hr>.

- Print a horizontal rule:

```bash
hr
```

- Print a horizontal rule with a custom string:

```bash
hr string
```

- Print a multiline horizontal rule:

```bash
hr string_a string_b string_c
```
