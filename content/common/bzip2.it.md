> Maggiori informazioni: <http://bzip.org>.

- Comprimi un file:

```bash
bzip2 percorso/al/file
```

- Decomprimi un file:

```bash
bzip2 -d percorso/al/file_compresso.bz2
```

- Decomprimi un file e mostrane il contenuto su standard output:

```bash
bzip2 -dc percorso/al/file_compresso.bz2
```
