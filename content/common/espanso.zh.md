> 更多信息：<https://espanso.org>.

- 检查状态：

```bash
espanso status
```

- 编辑配置：

```bash
espanso edit config
```

- 从 hub 商店（<https://hub.espanso.org/>）安装一个软件包：

```bash
espanso install 软件包的名字
```

- 重新启动（安装软件包后需要，在失败的情况下很有用）：

```bash
espanso restart
```
