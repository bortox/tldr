> More information: <https://github.com/phpro/grumphp>.

- Register the Git hooks:

```bash
grumphp git:init
```

- Trigger the pre-commit hook manually:

```bash
grumphp git:pre-commit
```

- Check every versioned file:

```bash
grumphp run
```
