> 더 많은 정보: <https://manned.org/bg>.

- 가장 최근에 일시 중단된 작업을 재개하고 background에서 실행:

```bash
bg
```

- 특정 작업을 재개하고(`jobs -l` 를 사용하여 ID 가져오기) background에서 실행:

```bash
bg %job_id
```
