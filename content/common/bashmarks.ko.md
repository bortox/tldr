> 더 많은 정보: <https://github.com/huyng/bashmarks>.

- 사용 가능한 북마크 나열:

```bash
l
```

- 현재 디렉토리를 "bookmark_name"으로 저장:

```bash
s bookmark_name
```

- 북마크된 디렉토리로 이동:

```bash
g bookmark_name
```

- 북마크 된 디렉토리 내용 출력:

```bash
p bookmark_name
```

- 북마크 삭제:

```bash
d bookmark_name
```
