> 更多信息：<https://github.com/electron/asar>.

- 打包一个文件或目录：

```bash
asar pack 文件或目录路径 输出的 asar 文件
```

- 解压一个 asar 文件：

```bash
asar extract asar 文件
```

- 从 asar 文件中解压指定的文件：

```bash
asar extract-file asar 文件 文件
```

- 列出一个 asar 文件中的内容：

```bash
asar list asar 文件
```
