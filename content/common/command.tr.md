> Daha fazla bilgi: <https://manned.org/command>.

- `ls` programını aynı isimde bir alias olsa dahi çalıştır:

```bash
command ls
```

- Alias'a atanan özel komutu veya çalıştırılabilir dosyanın yolunu göster:

```bash
command -v komut_ismi
```
