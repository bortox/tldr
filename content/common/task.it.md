> Maggiori informazioni: <https://manned.org/task>.

- Aggiungere un nuovo task:

```bash
task add thing_to_do
```

- Lista dei task:

```bash
task list
```

- Contrassegnare un task come completato:

```bash
task task_id done
```

- Modificare un task:

```bash
task task_id modify new_thing_to_do
```

- Eliminare un task:

```bash
task task_id delete
```
