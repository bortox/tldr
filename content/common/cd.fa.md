> اطلاعات بیشتر: <https://manned.org/cd>.

- تغییر مسیربه دایرکتوری مشخص شده:

```bash
cd path/to/directory
```

- تغییر مسیر به دایرکتوری Home کاربر فعلی:

```bash
cd
```

- تغییر مسیر به دایرکتوری بالاتر از مسیر جاری:

```bash
cd ..
```

- تغییر مسیر به دایرکتوری قبلی:

```bash
cd -
```
