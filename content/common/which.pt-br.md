> Mais informações: <https://manned.org/which>.

- Procura na variável PATH e exibe a localização do executável se encontrado:

```bash
which executável
```

- Se existe múltiplos executáveis que concidem com a pesquisa, exibe todos:

```bash
which -a executavél
```
