> 더 많은 정보: <https://www.clementine-player.org>.

- Clementine 열기:

```bash
clementine
```

- 음악 파일을 실행합니다:

```bash
clementine url_혹은_경로/파일.ext
```

- 정지, 재생 버튼(Toggle):

```bash
clementine --play-pause
```

- 재생 멈추기:

```bash
clementine --stop
```

- 다음 트랙 넘기기:

```bash
clementine --next
```

- 이전 트랙 넘기기:

```bash
clementine --previous
```

- 플레이리스트 파일 불러오기:

```bash
clementine --load 경로/재생리스트.ext
```

- 현재 불러온 플레이리스트에서 5번째 트랙 재생하기:

```bash
clementine --play-track 5
```
