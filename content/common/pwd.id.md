> Informasi lebih lanjut: <https://www.gnu.org/software/coreutils/pwd>.

- Mencetak direktori saat ini:

```bash
pwd
```

- Mencetak direktori saat ini, dan menjelaskan semua tautan simbolis (dengan kata lain menampilkan alamat fisik):

```bash
pwd -P
```
