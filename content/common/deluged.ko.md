> 더 많은 정보: <https://deluge-torrent.org>.

- Deluge 데몬 시작하기:

```bash
deluged
```

- 특정 포트에서 Deluge 데몬 시작하기:

```bash
deluged -p 포트번호
```

- 특정 구성파일을 이용하여 Deluge 데몬 시작하기:

```bash
deluged -c 경로/구성_파일명
```

- Deluge 데몬을 시작하고 파일에 로그 출력하기:

```bash
deluged -l 경로/로그_파일명
```
