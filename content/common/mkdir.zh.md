> 更多信息：<https://www.gnu.org/software/coreutils/mkdir>.

- 在当前目录或给定路径下创建一个目录：

```bash
mkdir 目录
```

- 在当前目录下创建多个目录：

```bash
mkdir 目录_1 目录_2 ...
```

- 递归地创建目录（对创建嵌套目录很有用）：

```bash
mkdir -p 路径/到/目录
```
