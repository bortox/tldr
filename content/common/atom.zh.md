> 由 `apm` 管理插件。

> 更多信息：<https://atom.io/>.

- 打开文件或目录：

```bash
atom path/to/file_or_directory
```

- 在新窗口中打开文件或目录：

```bash
atom -n path/to/file_or_directory
```

- 在已有窗口中打开文件或目录：

```bash
atom --add path/to/file_or_directory
```

- 以安全模式启动 Atom（不加载额外插件）：

```bash
atom --safe
```

- 在终端前台运行 Atom：

```bash
atom --foreground
```
