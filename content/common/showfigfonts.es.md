> Véase también `figlet`.

> Más información: <https://manned.org/showfigfonts>.

- Muestra las fuentes disponibles:

```bash
showfigfonts
```

- Muestra las fuentes disponibles usando un texto específico:

```bash
showfigfonts texto_de_entrada
```
