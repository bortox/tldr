> More information: <https://ninja-build.org/manual.html>.

- Build in the current directory:

```bash
ninja
```

- Build a program in a given directory:

```bash
ninja -C path/to/directory
```

- Show targets (e.g. `install` and `uninstall`):

```bash
ninja -t targets
```

- Show help:

```bash
ninja -h
```
