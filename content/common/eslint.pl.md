> Więcej informacji: <https://eslint.org>.

- Stwórz eslint config:

```bash
eslint --init
```

- Lint na danym zestawie plików:

```bash
eslint nazwapliku.js nazwapliku1.js
```

- Napraw lint issues:

```bash
eslint --fix
```

- Lint z config:

```bash
eslint -c sciezka/do/pliku_config app/src
```
