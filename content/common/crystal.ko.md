> 더 많은 정보: <https://crystal-lang.org/reference/using_the_compiler>.

- Crystal 파일 실행:

```bash
crystal 경로/파일.cr
```

- 단일 실행 파일로의 종속성 및 파일 컴파일:

```bash
crystal build 경로/파일.cr
```

- 언어 테스트를 위한 로컬 대화형 서버 시작:

```bash
crystal play
```

- Cystal 응용 프로그램을 위한 프로젝트 디렉토리 생성:

```bash
crystal init app 어플리케이션명
```

- 도움말 옵션 표시:

```bash
crystal help
```
