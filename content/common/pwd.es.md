> Más información: <https://www.gnu.org/software/coreutils/pwd>.

- Muestra el directorio actual:

```bash
pwd
```

- Muestra el directorio actual y resuelve todos los enlaces simbólicos (es decir, muestra la ruta "física"):

```bash
pwd -P
```
