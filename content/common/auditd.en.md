> It should not be invoked manually.

> More information: <https://manned.org/auditd>.

- Start the daemon:

```bash
auditd
```

- Start the daemon in debug mode:

```bash
auditd -d
```

- Start the daemon on-demand from launchd:

```bash
auditd -l
```
