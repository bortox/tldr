> More information: <http://elinks.or.cz>.

- Start elinks:

```bash
elinks
```

- Quit elinks:

```bash
Ctrl + C
```

- Dump output of webpage to console, colorizing the text with ANSI control codes:

```bash
elinks -dump -dump-color-mode 1 url
```
