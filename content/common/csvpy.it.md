> Incluso in csvkit.

> Maggiori informazioni: <https://csvkit.readthedocs.io/en/latest/scripts/csvpy.html>.

- Carica un file CSV in un oggetto `CSVKitReader`:

```bash
csvpy data.csv
```

- Carica un file CSV in un oggetto `CSVKitDictReader`:

```bash
csvpy --dict data.csv
```
