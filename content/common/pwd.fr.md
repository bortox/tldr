> Plus d'informations : <https://www.gnu.org/software/coreutils/pwd>.

- Affiche le répertoire actuel :

```bash
pwd
```

- Affiche le répertoire actuel tout en traduisant les liens symboliques (c.-à-d. afficher le répertoire « physique ») :

```bash
pwd -P
```
