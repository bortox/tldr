> See also: `gifsicle`.

> More information: <https://www.lcdf.org/gifsicle>.

- Check how GIFs differ:

```bash
gifdiff path/to/first.gif path/to/second.gif
```

- Check if GIFs differ:

```bash
gifdiff --brief path/to/first.gif path/to/second.gif
```
