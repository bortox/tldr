> 더 많은 정보: <http://bzip.org>.

- 파일 압축하기:

```bash
bzip2 경로/압축할_파일명
```

- 파일 압축해제하기:

```bash
bzip2 -d 경로/압축된_.bz2파일
```

- 파일을 표준 출력으로 압축해제:

```bash
bzip2 -dc 경로/압축된_.bz2파일
```
