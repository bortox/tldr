> Part of `git-extras`.

> More information: <https://github.com/tj/git-extras/blob/master/Commands.md#git-browse>.

- Open the first upstream in the default browser:

```bash
git browse
```

- Open a specific upstream in the default browser:

```bash
git browse upstream
```
