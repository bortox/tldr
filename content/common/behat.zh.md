> 更多信息：<https://behat.org>.

- 初始化一个 PHP behat 项目：

```bash
behat --init
```

- 运行所有测试：

```bash
behat
```

- 运行指定组所有的测试用例：

```bash
behat --suite=组名
```

- 运行所有测试，指定输入格式：

```bash
behat --format pretty|progress
```

- 将测试结果输出到指定文件：

```bash
behat --out 路径/到/文件
```

- 展示测试组所在的目录清单：

```bash
behat --definitions
```
