> More information: <https://github.com/google/zopfli>.

- Optimize a PNG:

```bash
zopflipng input.png output.png
```

- Optimize several PNGs and save with given prefix:

```bash
zopflipng --prefix=prefix image1.png image2.png image3.png
```
