> More information: <https://git-scm.com/docs/git-reflog>.

- Show the reflog for HEAD:

```bash
git reflog
```

- Show the reflog for a given branch:

```bash
git reflog branch_name
```

- Show only the 5 latest entries in the reflog:

```bash
git reflog -n 5
```
