> Maggiori informazioni: <https://fph.altervista.org/prog/bastet.html>.

- Invia un gioco tetris:

```bash
bastet
```

- Sposta il pezzo a sinistra o a destra:

```bash
Sinistra|Destra freccia
```

- Ruota in senso orario o inverso:

```bash
Spazio|tasto freccia su
```

- Caduta lenta:

```bash
tasto freccia gi√π
```

- Caduta veloce:

```bash
Invio
```

- Pausa:

```bash
p
```

- Lasciare il gioco:

```bash
Ctrl + C
```
