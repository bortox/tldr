> See also `write`.

> More information: <https://manned.org/mesg>.

- Check terminal's openness to write messages:

```bash
mesg
```

- Disable receiving messages from the write command:

```bash
mesg n
```

- Enable receiving messages from the write command:

```bash
mesg y
```
