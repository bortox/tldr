> 더 많은 정보: <https://github.com/electron/asar>.

- 파일 또는 디렉토리 보관:

```bash
asar pack path/to/file_or_directory archived.asar
```

- 보관소 추출:

```bash
asar extract archived.asar
```

- 보관소에서 특정 파일 추출:

```bash
asar extract-file archived.asar file
```

- 보관소 파일의 내용을 나열:

```bash
asar list archived.asar
```
