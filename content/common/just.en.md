> More information: <https://github.com/casey/just>.

- Run a recipe specified in the justfile:

```bash
just recipe
```

- Initialize new justfile in project root:

```bash
just --init
```

- Edit justfile in the default editor:

```bash
just -e
```

- List available recipes in the justfile:

```bash
just -l
```

- Print justfile:

```bash
just --dump
```
