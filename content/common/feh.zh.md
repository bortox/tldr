> 更多信息：<https://feh.finalrewind.org>.

- 查看本地图像或使用 URL：

```bash
feh 图片路径
```

- 递归查看图像：

```bash
feh --recursive 图片路径
```

- 使用无边框窗口查看图像：

```bash
feh --borderless 图片路径
```

- 在浏览完最后一个图像之后退出：

```bash
feh --cycle-once 图片路径
```

- 设置幻灯片放映周期延迟时间（秒）：

```bash
feh --slideshow-delay 秒 图片路径
```

- 设置墙纸（居中、填充、最大化、缩放或平铺）：

```bash
feh --bg-center|fill|max|scale|tile 图片路径
```
