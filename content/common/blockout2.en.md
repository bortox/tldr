> More information: <http://www.blockout.net/blockout2/>.

- Start a new game:

```bash
blockout2
```

- Navigate the current piece on a 2D plane:

```bash
Up|Down|Left|Right arrow key
```

- Rotate the piece on its axis:

```bash
Q|W|E|A|S|D
```

- Hard drop the current piece:

```bash
Spacebar
```

- Pause/unpause the game:

```bash
p
```
