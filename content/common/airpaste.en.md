> More information: <https://github.com/mafintosh/airpaste>.

- Wait for a message and display it when received:

```bash
airpaste
```

- Send text:

```bash
echo text | airpaste
```

- Send a file:

```bash
airpaste < path/to/file
```

- Receive a file:

```bash
airpaste > path/to/file
```

- Create or join a channel:

```bash
airpaste channel_name
```
