> 더 많은 정보: <https://manned.org/cd>.

- 주어진 디렉토리로 이동:

```bash
cd 경로/디렉토리명
```

- 현재 사용자의 홈 디렉토리로 이동:

```bash
cd
```

- 현재 디렉토리의 상위 디렉토리로 이동:

```bash
cd ..
```

- 이전에 선택되었던 디렉토리로 이동:

```bash
cd -
```
