> 更多信息：<https://manned.org/cd>.

- 转到指定目录：

```bash
cd 路径/到/目录
```

- 转到当前目录的父目录：

```bash
cd ..
```

- 转到当前用户的主（home）目录：

```bash
cd
```

- 转到指定用户的主（home）目录：

```bash
cd ~用户名
```

- 转到刚才选择的目录：

```bash
cd -
```
