> Use arrow keys to navigate. Right and Left to view and return from a submission, respectively.

> More information: <https://github.com/michael-lazar/rtv>.

- Open the front page:

```bash
/front
```

- Open a subreddit:

```bash
/r/subreddit_name
```

- Expand/collapse comments:

```bash
[space]
```

- Open link:

```bash
o
```

- Log in:

```bash
u
```

- Open the help screen:

```bash
?
```
