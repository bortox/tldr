> अधिक जानकारी: <https://manned.org/cd>।

- दी गई निर्देशिका पर जाएं:

```bash
cd निर्देशिका / का / पथ
```

- वर्तमान उपयोगकर्ता की होम निर्देशिका पर जाएं:

```bash
cd
```

- वर्तमान निर्देशिका के जनक तक जाएं:

```bash
cd ..
```

- पहले चुनी गई निर्देशिका पर जाएं:

```bash
cd -
```
