> 更多信息：<https://virtualenv.pypa.io/>.

- 创建新环境：

```bash
virtualenv path/to/venv
```

- 自定义提示符：

```bash
virtualenv --prompt=prompt_prefix path/to/venv
```

- 为虚拟环境使用不同的 Python 版本：

```bash
virtualenv --python=path/to/pythonbin path/to/venv
```

- 启动（选择）环境：

```bash
source path/to/venv/bin/activate
```

- 停止环境：

```bash
deactivate
```
