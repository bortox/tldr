> 更多信息：<https://github.com/rfc1036/whois>.

- 获取域名信息：

```bash
whois example.com
```

- 获取 IP 地址信息：

```bash
whois 8.8.8.8
```

- 获取 IP 地址用于报告滥用的联系方式：

```bash
whois -b 8.8.8.8
```
