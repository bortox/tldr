> 更多信息：<https://github.com/mafintosh/airpaste>.

- 等待接收消息并显示接收到的信息：

```bash
airpaste
```

- 发送文本：

```bash
echo 文本 | airpaste
```

- 发送文件：

```bash
airpaste < 文件的路径
```

- 接收文件：

```bash
airpaste > 文件的路径
```

- 创建 / 加入频道：

```bash
airpaste 频道名
```
