> More information: <https://traefik.io>.

- Start server with default config:

```bash
traefik
```

- Start server with a custom config file:

```bash
traefik --c config_file.toml
```

- Start server with cluster mode enabled:

```bash
traefik --cluster
```

- Start server with web UI enabled:

```bash
traefik --web
```
