> 更多信息：<https://hexo.io/>.

- 初始化一个网站：

```bash
hexo init path/to/directory
```

- 创建一篇新文章：

```bash
hexo new layout title
```

- 构建静态文件：

```bash
hexo generate
```

- 启动本地服务器：

```bash
hexo server
```

- 部署网站：

```bash
hexo deploy
```

- 清理缓存文件（`db.json`）和生成的文件（`public/`）：

```bash
hexo clean
```
