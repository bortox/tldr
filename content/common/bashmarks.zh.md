> 更多信息：<https://github.com/huyng/bashmarks>.

- 可访问书签清单：

```bash
l
```

- 保存当前目录到某书签里：

```bash
s 书签名
```

- 跳转到指定书签：

```bash
g 书签名
```

- 打印书签目录内容：

```bash
p 书签名
```

- 删除书签：

```bash
d 书签名
```
