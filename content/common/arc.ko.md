> 더 많은 정보: <https://secure.phabricator.com/book/phabricator/article/arcanist/>.

- 검토에 대한 차이점의 변경을 보냅니다:

```bash
arc diff
```

- 보류중인 수정 정보 표시:

```bash
arc list
```

- 검토 후 Git Commit 메시지 업데이트하기:

```bash
arc amend
```

- Git에 Push 하기:

```bash
arc land
```
