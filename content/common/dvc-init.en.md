> More information: <https://dvc.org/doc/command-reference/init>.

- Initialize a new local repository:

```bash
dvc init
```

- Initialize DVC without Git:

```bash
dvc init --no-scm
```

- Initialize DVC in a subdirectory:

```bash
cd path/to/subdir && dvc init --sudir
```
