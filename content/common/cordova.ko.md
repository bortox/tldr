> 더 많은 정보: <https://cordova.apache.org/docs/en/latest/guide/cli/>.

- cordova 프로젝트 생성:

```bash
cordova create 경로 패키지_명 프로젝트_명
```

- 현재 작업 공간 상태 표시:

```bash
cordova info
```

- cordova 플랫폼 추가:

```bash
cordova platform add 플랫폼
```

- cordova 플랫폼 제거:

```bash
cordova platform remove 플랫폼
```

- cordova 플러그인 추가:

```bash
cordova plugin add 플러그인id
```

- cordova 플러그인 제거:

```bash
cordova plugin remove 플러그인id
```
