> More information: <https://eslint.org>.

- Create ESLint config:

```bash
eslint --init
```

- Lint on a given set of files:

```bash
eslint filename.js filename1.js
```

- Fix lint issues:

```bash
eslint --fix
```

- Lint with config:

```bash
eslint -c path/to/config_file app/src
```
