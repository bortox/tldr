> 此命令也有关于其子命令的文件，例如：`amass db`.

> 更多信息：<https://github.com/OWASP/Amass>.

- 查看 Amass 版本：

```bash
amass -version
```

- 展示帮助信息：

```bash
amass -help
```

- 展示子命令帮助信息（如 `intel`、`enum` 等）：

```bash
amass -help 子命令
```

- 执行 Amass 子命令：

```bash
amass 子命令
```
