> 更多信息：<http://www.brain-dump.org/projects/abduco/>.

- 列出会话：

```bash
abduco
```

- 附加到一个会话，如果不存在则新建它：

```bash
abduco -A 会话名 终端
```

- 使用`dvtm`附加到一个会话，如果不存在则新建它：

```bash
abduco -A 会话名
```

- 从一个会话中分离：

```bash
Ctrl + \
```

- 以只读模式附加到一个会话：

```bash
abduco -Ar 会话名
```
