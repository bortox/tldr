> 更多信息：<https://www.jenv.be/>.

- 向 jEnv 添加一个 Java 版本：

```bash
jenv add Java Home 路径
```

- 显示当前使用的 JDK 版本：

```bash
jenv version
```

- 显示所有托管的 JDK：

```bash
jenv versions
```

- 设置全局JDK版本：

```bash
jenv global Java 版本
```

- 设置当前 shell 会话的 JDK 版本：

```bash
jenv shell Java 版本
```

- 启用 jEnv 插件：

```bash
jenv enable-plugin 插件名称
```
