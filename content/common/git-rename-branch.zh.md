> 属于 `git-extras` 的一部分。

> 更多信息：<https://github.com/tj/git-extras/blob/master/Commands.md#git-rename-branch>.

- 重新命名你目前所在的分支：

```bash
git rename-branch 新的分支名称
```

- 重新命名一个特定的分支：

```bash
git rename-branch 旧的分支名称 新的分支名称
```
