> More information: <https://docs.platformio.org/en/latest/core/userguide/cmd_upgrade.html>.

- Update PlatformIO to the latest version:

```bash
pio upgrade
```

- Update PlatformIO to the latest development (unstable) version:

```bash
pio upgrade --dev
```
