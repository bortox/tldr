> 参见 `atom`.

> 更多信息：<https://github.com/atom/apm>.

- 下载包：http://atom.io/packages 和主题 http://atom.io/themes：

```bash
apm install 包名
```

- 移除包 / 主题：

```bash
apm remove 包名
```

- 升级包 / 主题：

```bash
apm upgrade 包名
```
