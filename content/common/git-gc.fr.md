> Plus d'informations : <https://git-scm.com/docs/git-gc>.

- Optimise le dépôt :

```bash
git gc
```

- Optimise le dépôt plus agressivement, plus long :

```bash
git gc --aggressive
```

- Afficher les objets à supprimer :

```bash
git gc --no-prune
```

- Supprime tout les objets trouvés sans les afficher sur la sortie standard :

```bash
git gc --quiet
```

- Afficher le manuel :

```bash
git gc --help
```
