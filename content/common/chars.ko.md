> 더 많은 정보: <https://github.com/antifuchs/chars>.

- 밸류 값으로 문자 검색:

```bash
chars 'ß'
```

- 유니코드 코드로 문자 검색:

```bash
chars U+1F63C
```

- 모호한 코드 포인트가 주어지면 가능한 문자 검색:

```bash
chars 10
```

- 제어 문자 찾기:

```bash
chars "^C"
```
