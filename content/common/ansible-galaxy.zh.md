> 更多信息：<https://docs.ansible.com/ansible/latest/cli/ansible-galaxy.html>.

- 安装一个角色：

```bash
ansible-galaxy install 用户名.角色名
```

- 移除一个角色：

```bash
ansible-galaxy remove 用户名.角色名
```

- 列出已安装的角色：

```bash
ansible-galaxy list
```

- 搜索一个指定的角色：

```bash
ansible-galaxy search 角色名
```

- 创建一个新的角色：

```bash
ansible-galaxy init 角色名
```
