> 更多資訊：<https://tldr.sh>.

- 顯示指令的常用用法：

```bash
tldr 指令
```

- 顯示 Linux 作業系統中 tar 指令的用法：

```bash
tldr -p linux tar
```

- 顯示 Git 的子指令 checkout 的用法：

```bash
tldr git-checkout
```
