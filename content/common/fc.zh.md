> 更多信息：<https://manned.org/fc>.

- 在系统默认编辑器中打开：

```bash
fc
```

- 指定要使用的的编辑器：

```bash
fc -e 'emacs'
```

- 从历史记录中列出最近的命令：

```bash
fc -l
```
