> 더 많은 정보: <https://graphviz.org/doc/info/command.html>.

- 입력 파일 이름과 선택한 포맷을 기반으로 이미지 파일을 랜더링하고 출력파일 이름 결정하기:

```bash
dot -Tpng -O 경로/파일명.dot
```

- DOT 파일로부터 SVG 생성하기:

```bash
dot -Tsvg -o 경로/출력_파일명.svg 경로/파일명.dot
```
