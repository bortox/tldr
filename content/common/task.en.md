> More information: <https://manned.org/task>.

- Add new task:

```bash
task add thing_to_do
```

- List tasks:

```bash
task list
```

- Mark task as completed:

```bash
task task_id done
```

- Modify task:

```bash
task task_id modify new_thing_to_do
```

- Delete task:

```bash
task task_id delete
```
