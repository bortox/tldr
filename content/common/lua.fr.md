> Plus d'informations : <https://www.lua.org>.

- Démarre une session de commandes interactive Lua :

```bash
lua
```

- Exécute un script Lua :

```bash
lua nom_du_script.lua --arguments-facultatifs
```

- Exécute une expression Lua :

```bash
lua -e 'print( "Hello World" )'
```
