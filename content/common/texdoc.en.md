> More information: <https://texdoc.org/index.html>.

- Open the first search result in the default PDF viewer:

```bash
texdoc search
```

- List the best search results:

```bash
texdoc --list search
```

- Open full documentation of texdoc:

```bash
texdoc texdoc
```
