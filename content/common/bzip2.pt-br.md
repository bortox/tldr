> Mais informações: <http://bzip.org>.

- Compactar um arquivo:

```bash
bzip2 arquivo
```

- Descompactar um arquivo:

```bash
bzip2 -d arquivo.bz2
```

- Descompactar um arquivo exibindo o conteúdo no terminal:

```bash
bzip2 -dc arquivo.bz2
```
