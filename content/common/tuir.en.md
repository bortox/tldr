> Navigate with the Vim keys.

> More information: <https://gitlab.com/ajak/tuir>.

- Launch tuir:

```bash
tuir
```

- Open a subreddit:

```bash
/subreddit_name
```

- Open a link:

```bash
o
```

- Open a specific subreddit on launch:

```bash
tuir -s subreddit_name
```

- Open external links using programs defined in the mailcap config:

```bash
tuir --enable-media
```
