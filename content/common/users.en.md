> See also: `useradd`, `userdel`, `usermod`.

> More information: <https://www.gnu.org/software/coreutils/users>.

- Print logged in usernames:

```bash
users
```

- Print logged in usernames according to a given file:

```bash
users /var/log/wmtp
```
