> More information: <https://myrepos.branchable.com>.

- Register a repository:

```bash
mr register
```

- Update repositories in 5 concurrent jobs:

```bash
mr -j5 update
```

- Print the status of all repositories:

```bash
mr status
```

- Checkout all repositories to the latest version:

```bash
mr checkout
```
