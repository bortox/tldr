> More information: <https://thunderbird.net>.

- Open thunderbird:

```bash
thunderbird
```

- Use a specific user profile:

```bash
thunderbird -P profile_name
```

- Use a specific user profile directory:

```bash
thunderbird --profile path/to/profile/directory
```
