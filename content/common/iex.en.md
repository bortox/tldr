> More information: <https://hexdocs.pm/iex>.

- Start an interactive session:

```bash
iex
```

- Start a session that remembers history:

```bash
iex --erl "-kernel shell_history enabled"
```

- Start and load Mix project files:

```bash
iex -S mix
```
