> 더 많은 정보: <https://manned.org/apropos>.

- 키워드 검색:

```bash
apropos 정규 표현식
```

- 출력을 터미널 너비에 제한을 두지 않고 검색:

```bash
apropos -l 정규 표현식
```

- 주어진 모든 표현식만 포함하는 페이지 검색(AND 검색):

```bash
apropos 정규 표현식_1 -a 정규 표현식_2 -a 정규 표현식_3
```
