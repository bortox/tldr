> `atom`을 보시오.

> 더 많은 정보: <https://github.com/atom/apm>.

- <http://atom.io/packages>으로부터 패키지 설치하고 <http://atom.io/themes>에서 테마 설치:

```bash
apm install 패키지 이름
```

- 패키지/테마 제거:

```bash
apm remove 패키지 이름
```

- 패키지/테마 업그레이드:

```bash
apm upgrade 패키지 이름
```
