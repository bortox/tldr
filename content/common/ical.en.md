> More information: <https://manned.org/ical>.

- Display the current month's calendar:

```bash
ical
```

- Convert a Gregorian date to a Hijri date:

```bash
ical --gregorian yyyymmdd
```

- Convert a Hirji date to a Gregorian date:

```bash
ical --hijri yyyymmdd
```
