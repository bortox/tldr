> Mais informações: <https://cowlark.com/wordgrinder>.

- Iniciar o wordgrinder (carrega um documento vazio por padrão):

```bash
wordgrinder
```

- Abrir um arquivo específico:

```bash
wordgrinder nome_do_arquivo
```

- Mostrar o menu:

```bash
Alt + M
```
