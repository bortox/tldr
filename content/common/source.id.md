> Informasi lebih lanjut: <https://manned.org/source>.

- Mengevaluasi konten pada file yang ditentukan:

```bash
source jalan/menuju/file
```

- Mengevaluasi konten pada file yang ditentukan (`.` adalah alias dari `source`)

```bash
. jalan/menuju/file
```
