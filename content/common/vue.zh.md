> 此命令也有关于其子命令的文件，例如：`vue build`.

> 更多信息：<https://cli.vuejs.org>.

- 从模板创建新项目：

```bash
vue init template project_name
```

- 从本地模板创建新项目：

```bash
vue init path/to/template_directory project_name
```

- 从 GitHub 上拉取模板并创建新项目：

```bash
vue init username/repo project_name
```
