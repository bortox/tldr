> Maggiori informazioni: <https://github.com/github/git-sizer>.

- Mostra solo le statistiche che hanno un livello di rischio maggiore di 0:

```bash
git sizer
```

- Mostra tutte le statistiche:

```bash
git sizer -v
```

- Mostra opzioni aggiuntive:

```bash
git sizer -h
```
