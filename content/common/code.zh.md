> 更多信息：<https://github.com/microsoft/vscode>.

- 打开 VS Code：

```bash
code
```

- 在 VS Code 中打开当前目录：

```bash
code .
```

- 在 VS Code 打开一个文件或目录：

```bash
code 路径/文件或目录
```

- 在当前打开的 VS Code 窗口中打开一个文件或目录：

```bash
code --reuse-window 路径/文件或目录
```

- 在 VS Code 中对比两个文件：

```bash
code -d 文件1 文件2
```

- 用超级用户（sudo）权限打开 VS Code：

```bash
sudo code 路径/文件或目录 --user-data-dir
```
