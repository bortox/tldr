> 더 많은 정보: <https://asdf-vm.com>.

- 사용 가능한 모든 플러그인을 나열:

```bash
asdf plugin-list-all
```

- 플러그인 설치:

```bash
asdf plugin-add 이름
```

- 모든 사용 가능한 패키지 버전 나열:

```bash
asdf list-all 이름
```

- 특정 패키지 버전 설치:

```bash
asdf install 이름 버전
```

- 글로벌 버전 패키지 설치:

```bash
asdf global 버전 버전
```

- 로컬 버전 패키지 설치:

```bash
asdf local 이름 버전
```
