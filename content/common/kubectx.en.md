> More information: <https://github.com/ahmetb/kubectx>.

- List the contexts:

```bash
kubectx
```

- Switch to a named context:

```bash
kubectx name
```

- Switch to the previous context:

```bash
kubectx -
```

- Delete a named context:

```bash
kubectx -d name
```
