> More information: <http://www.offlineimap.org>.

- Synchronize once, without enabling autorefresh:

```bash
offlineimap -o
```

- Synchronize a specific account:

```bash
offlineimap -a account
```

- Synchronize a specific folder:

```bash
offlineimap -f folder
```
