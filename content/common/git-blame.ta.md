> மேலும் விவரத்திற்கு: <https://git-scm.com/docs/git-blame>.

- எழுத்தாளர் பெயருடன் கோப்பை அச்சிட்டு ஒவ்வொரு வரியிலும் ஹாஷ் செய்யுங்கள்:

```bash
git blame கோப்பு
```

- எழுத்தாளர் மின்னஞ்சலுடன் கோப்பை அச்சிட்டு ஒவ்வொரு வரியிலும் ஹாஷ் செய்யுங்கள்:

```bash
git blame -e கோப்பு
```
