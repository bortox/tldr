> Mais informações: <https://github.com/busterc/xcv>.

- Cortar um arquivo:

```bash
xcv x arquivo_a_ser_cortado
```

- Copiar um arquivo:

```bash
xcv c arquivo_a_ser_copiado
```

- Colar um arquivo:

```bash
xcv v arquivo_a_ser_colado
```

- Listar todos os arquivos disponíveis para serem colados:

```bash
xcv l
```
