> Some subcommands such as `hg commit` have their own usage documentation.

> More information: <https://www.mercurial-scm.org>.

- Execute Mercurial command:

```bash
hg command
```

- Call general help:

```bash
hg help
```

- Call help on a command:

```bash
hg help command
```

- Check the Mercurial version:

```bash
hg --version
```
