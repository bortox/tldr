> Plus d'informations : <https://github.com/tj/git-extras/blob/master/Commands.md#git-ignore-io>.

- Lister les templates disponibles :

```bash
git ignore-io list
```

- Générer un template .gitignore :

```bash
git ignore-io item_a,item_b,item_n
```
