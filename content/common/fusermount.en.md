> More information: <https://manned.org/fusermount>.

- Unmount a FUSE filesystem:

```bash
fusermount -u path/to/mount_point
```

- Unmount a FUSE filesystem as soon as it becomes unused:

```bash
fusermount -z path/to/mount_point
```

- Display version:

```bash
fusermount --version
```
