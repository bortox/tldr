> More information: <https://uk.mathworks.com/help/matlab/>.

- Run without splash screen during startup:

```bash
matlab -nosplash
```

- Execute a MATLAB statement:

```bash
matlab -r "matlab_statement"
```

- Run a MATLAB script:

```bash
matlab -r "run(path/to/script.m)"
```
