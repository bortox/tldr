> More information: <http://bzip.org>.

- Compress a file:

```bash
bzip2 path/to/file_to_compress
```

- Decompress a file:

```bash
bzip2 -d path/to/compressed_file.bz2
```

- Decompress a file to standard output:

```bash
bzip2 -dc path/to/compressed_file.bz2
```
