> More information: <https://github.com/mtoyoda/sl>.

- Let a steam locomotive run through your terminal:

```bash
sl
```

- The train burns, people scream:

```bash
sl -a
```

- Let the train fly:

```bash
sl -F
```

- Make the train little:

```bash
sl -l
```

- Let the user exit (CTRL + C):

```bash
sl -e
```
