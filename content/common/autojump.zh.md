> 使用 `j`、`jc`、和 `jo` 作为别名。

> 更多信息：<https://github.com/wting/autojump>.

- 跳转到包含指定通配符的目录：

```bash
j 通配符表达式
```

- 跳转到包含指定通配符的目录的下一级：

```bash
jc 通配符表达式
```

- 使用系统文件管理器，打开指定的目录：

```bash
jo 通配符表达式
```

- 从 autojump 数据库中删除不存在的目录：

```bash
j --purge
```

- 展示 autojump 数据库数据：

```bash
j -s
```
