> 更多信息：<https://nodejs.org>.

- 运行一个 JavaScript 文件：

```bash
node 文件名
```

- 开始一个 REPL 交互式解释器：

```bash
node
```

- 执行输入的 JavaScript 代码：

```bash
node -e "代码"
```

- 执行输入的 JavaScript 代码并显示结果：

```bash
node -p "代码"
```

- 启动检查器并在程序源码解析完成后等待调试器连接：

```bash
node --no-lazy --inspect-brk 文件名
```
