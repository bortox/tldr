> 更多信息：<https://uk.mathworks.com/help/matlab/>.

- 在启动过程中，运行时不出现闪屏：

```bash
matlab -nosplash
```

- 执行 MATLAB 语句：

```bash
matlab -r "matlab_语句"
```

- 运行 MATLAB 脚本：

```bash
matlab -r "run(路径/script.m)"
```
