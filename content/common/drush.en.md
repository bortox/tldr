> More information: <https://www.drush.org>.

- Enable module "foo":

```bash
drush en foo
```

- Uninstall module "foo":

```bash
drush pmu foo
```

- Clear all caches:

```bash
drush cr
```

- Clear CSS and JavaScript caches:

```bash
drush cc css-js
```
