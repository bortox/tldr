> Weitere Informationen: <https://www.gnu.org/software/coreutils/sleep>.

- Verzögere in Sekunden:

```bash
sleep sekunden
```

- Verzögere in Minuten:

```bash
sleep minutenm
```

- Verzögere in Stunden:

```bash
sleep stundenh
```
