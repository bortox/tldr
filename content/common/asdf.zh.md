> 更多信息：<https://asdf-vm.com>.

- 可用插件清单：

```bash
asdf plugin-list-all
```

- 安装插件：

```bash
asdf plugin-add 插件名
```

- 软件包的可用版本清单：

```bash
asdf list-all 软件包名
```

- 安装指定版本的软件包：

```bash
asdf install 软件包名 版本
```

- 设置软件包的全局安装版本：

```bash
asdf global 软件包名 版本
```

- 设置软件包的本地版本：

```bash
asdf local 软件包名 版本
```
