> `7z` と似ていますが、 `7zr` は `.7z` 形式のみをサポートしています。

> 詳しくはこちら: <https://www.7-zip.org>

- ファイルまたはディレクトリを圧縮する:

```bash
7zr a アーカイブ.7z ファイルまたはディレクトリへのパス
```

- 元のディレクトリ構造を保持したまま展開する:

```bash
7zr x アーカイブ.7z
```

- アーカイブの内容を表示する:

```bash
7zr l アーカイブ.7z
```
