> More information: <https://www.lua.org>.

- Start an interactive Lua shell:

```bash
lua
```

- Execute a Lua script:

```bash
lua script_name.lua --optional-argument
```

- Execute a Lua expression:

```bash
lua -e 'print( "Hello World" )'
```
