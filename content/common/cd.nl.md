> Meer informatie: <https://manned.org/cd>.

- Ga naar de gegeven map:

```bash
cd pad/naar/map
```

- Ga naar de thuismap van de huidige gebruiker:

```bash
cd
```

- Ga naar de ouder van de huidige map:

```bash
cd ..
```

- Ga naar de vorige map:

```bash
cd -
```
