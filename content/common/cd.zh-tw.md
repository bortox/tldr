> 更多資訊：<https://manned.org/cd>.

- 前往指定目錄：

```bash
cd 目錄/完整/路徑
```

- 前往目前使用者的家目錄：

```bash
cd
```

- 前往目前目錄的上層目錄：

```bash
cd ..
```

- 前往之前所在的目錄：

```bash
cd -
```
