> Informasi lebih lanjut: <https://laravel-zero.com>.

- Buat aplikasi Laravel Zero baru:

```bash
laravel-zero new nama
```

- Perbarui pemasang ke versi terkini:

```bash
laravel-zero self-update
```

- Lis perintah yang tersedia:

```bash
laravel-zero list
```
