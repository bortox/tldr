> More information: <https://github.com/mbrubeck/agate>.

- Run and generate a private key and certificate:

```bash
agate --content path/to/content/ --addr [::]:1965 --addr 0.0.0.0:1965 --hostname example.com --lang en-US
```

- Run server:

```bash
agate path/to/file
```

- Display help:

```bash
agate -h
```
