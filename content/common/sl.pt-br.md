> Mais informações: <https://github.com/mtoyoda/sl>.

- Faz uma locomotiva a vapor passar pelo seu terminal:

```bash
sl
```

- O trem pega fogo e pessoas gritam:

```bash
sl -a
```

- Faz o trem voar:

```bash
sl -F
```

- Faz o trem pequeno:

```bash
sl -l
```

- Deixa o usuário sair (CTRL + C):

```bash
sl -e
```
