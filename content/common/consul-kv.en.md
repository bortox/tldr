> More information: <https://learn.hashicorp.com/consul/getting-started/kv>.

- Read a value from the key-value store:

```bash
consul kv get key
```

- Store a new key-value pair:

```bash
consul kv put key value
```

- Delete a key-value pair:

```bash
consul kv delete key
```
