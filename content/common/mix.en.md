> More information: <https://hexdocs.pm/mix>.

- Execute a particular file:

```bash
mix run my_script.exs
```

- Create a new project:

```bash
mix new project_name
```

- Compile project:

```bash
mix compile
```

- Run project tests:

```bash
mix test
```

- List all mix commands:

```bash
mix help
```
