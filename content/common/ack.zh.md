> 更多信息：<https://beyondgrep.com/documentation>.

- 寻找包含"小明"的文件：

```bash
ack 小明
```

- 在给定文件类型中寻找包含"小明"的文件：

```bash
ack --ruby 小明
```

- 计算匹配到"小明"的总次数：

```bash
ack -ch 小明
```

- 列出内容包含"小明"的文件的文件名，并显示在每个文件中匹配的次数：

```bash
ack -cl 小明
```
