> More information: <https://geth.ethereum.org>.

- Connect to the main Ethereum network and automatically download the full node:

```bash
geth
```

- Connect to the Ropsten test network:

```bash
geth --testnet
```

- Create a new account:

```bash
geth account new
```

- Enable mining:

```bash
geth --mine
```
