> 更多信息：<https://github.com/tytso/pwgen>.

- 生成指定长度的随机密码：

```bash
pwgen -y 长度
```

- 生成安全、难以记忆的密码：

```bash
pwgen -s 长度
```

- 生成至少包含一个大写字母的密码：

```bash
pwgen -c 长度
```
