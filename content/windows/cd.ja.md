> 詳しくはこちら: <https://docs.microsoft.com/windows-server/administration/windows-commands/cd>

- 同じドライブ内のディレクトリに移動します:

```bash
cd ディレクトリパス
```

- 現在のディレクトリの名前を表示します:

```bash
cd
```

- 現在のディレクトリの親に移動します:

```bash
cd ..
```

- 別のドライブのディレクトリに移動します:

```bash
cd ディレクトリパス /d
```
