> 更多信息：<https://docs.microsoft.com/windows-server/administration/windows-commands/dir>.

- 显示当前目录的内容：

```bash
dir
```

- 显示指定目录的内容：

```bash
dir 目录的路径
```

- 显示当前目录的内容，包括隐藏的文件：

```bash
dir /A
```

- 显示指定目录的内容，包括隐藏的文件：

```bash
dir 目录的路径 /A
```
