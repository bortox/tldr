> 更多資訊：<https://docs.microsoft.com/windows-server/administration/windows-commands/dir>.

- 顯示當前目錄的內容：

```bash
dir
```

- 顯示指定目錄的內容：

```bash
dir 目錄/完整/路徑
```

- 顯示當前目錄的內容，包括隱藏的文件：

```bash
dir /A
```

- 顯示指定目錄的內容，包括隱藏的文件：

```bash
dir 目錄/完整/路徑 /A
```
