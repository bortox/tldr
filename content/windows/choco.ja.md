> `choco install` のようないくつかのサブコマンドには、使用方法についての独自のドキュメントがあります。

> 詳しくはこちら: <https://chocolatey.org>

- Chocolateyコマンドを実行します:

```bash
choco コマンド
```

- 一般的なヘルプを表示します:

```bash
choco -?
```

- 特定のコマンドのヘルプを表示します:

```bash
choco コマンド -?
```

- Chocolateyバージョンを表示します:

```bash
choco --version
```
