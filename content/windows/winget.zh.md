> 更多信息：<https://docs.microsoft.com/windows/package-manager/winget>.

- 安装一个软件包：

```bash
winget install package
```

- 显示关于一个软件包的信息：

```bash
winget show package
```

- 查找一个软件包：

```bash
winget search package
```
