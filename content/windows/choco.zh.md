> 此命令也有关于其子命令的文件，例如：`choco install`.

> 更多信息：<https://chocolatey.org>.

- 执行 Chocolatey 命令：

```bash
choco 命令
```

- 显示基本的帮助页面：

```bash
choco -?
```

- 显示指定命令的帮助页面：

```bash
choco 命令 -?
```

- 显示 Chocolatey 的版本：

```bash
choco --version
```
