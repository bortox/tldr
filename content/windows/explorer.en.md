> More information: <https://ss64.com/nt/explorer.html>.

- Open Windows Explorer:

```bash
explorer
```

- Open Windows Explorer in the current directory:

```bash
explorer .
```

- Open Windows Explorer in a specific directory:

```bash
explorer path/to/directory
```
