> 更多信息：<https://docs.microsoft.com/windows-server/administration/windows-commands/assoc>.

- 显示所有当前文件扩展名关联的列表：

```bash
assoc
```

- 显示指定扩展名的关联文件类型：

```bash
assoc .txt
```

- 将文件扩展名与特定的文件类型关联：

```bash
assoc .txt=txtfile
```
