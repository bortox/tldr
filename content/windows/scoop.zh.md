> 更多信息：<https://scoop.sh>.

- 安装一个包：

```bash
scoop install 包名
```

- 删除一个包：

```bash
scoop uninstall 包名
```

- 更新所有已安装的包：

```bash
scoop update *
```

- 列出所有已安装的包：

```bash
scoop list
```

- 显示一个包的信息：

```bash
scoop info 包名
```

- 搜索一个包：

```bash
scoop search 包名
```

- 移除所有包的旧版本并清理下载缓存：

```bash
scoop cleanup -k *
```
