> 此命令也有关于其子命令的文件，例如：`reg add`.

> 更多信息：<https://docs.microsoft.com/windows-server/administration/windows-commands/reg>.

- 执行注册表命令：

```bash
reg 命令
```

- 显示常规信息并列出所有可用命令：

```bash
reg /?
```

- 显示指定命令的帮助信息：

```bash
reg 命令 /?
```
