> 更多信息：<https://docs.microsoft.com/windows-server/administration/windows-commands/set>.

- 列出当前所有环境变量：

```bash
set
```

- 为一个环境变量设置指定的值：

```bash
set 变量名=值
```

- 列出以指定字符串开头的环境变量：

```bash
set 变量名
```

- 提示用户输入指定变量的值：

```bash
set /p 变量名=提示信息
```
