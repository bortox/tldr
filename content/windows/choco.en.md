> Some subcommands such as `choco install` have their own usage documentation.

> More information: <https://chocolatey.org>.

- Execute Chocolatey command:

```bash
choco command
```

- Call general help:

```bash
choco -?
```

- Call help on a specific command:

```bash
choco command -?
```

- Check the Chocolatey version:

```bash
choco --version
```
