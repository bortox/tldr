> 更多信息：<https://docs.microsoft.com/windows-server/administration/windows-commands/cipher>.

- 加密文件或目录：

```bash
cipher /e:路径/文件或目录
```

- 解密文件或目录：

```bash
cipher /d:路径/文件或目录
```

- 安全地删除文件或目录：

```bash
cipher /w:路径/文件或目录
```
