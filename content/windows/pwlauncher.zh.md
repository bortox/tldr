> 更多信息：<https://docs.microsoft.com/windows-server/administration/windows-commands/pwlauncher>.

- 显示当前 Windows To Go 的状态：

```bash
pwlauncher
```

- 启用或禁用 Windows To Go 的启动选项：

```bash
pwlauncher /enable|disable
```
