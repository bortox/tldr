> 詳しくはこちら: <https://chocolatey.org/docs/commands-feature>

- 利用可能な機能のリストを表示します:

```bash
choco feature list
```

- 機能を有効にします:

```bash
choco feature enable --name 名
```

- 機能を無効にします:

```bash
choco feature disable --name 名
```
