> 詳しくはこちら: <https://docs.microsoft.com/windows-server/administration/windows-commands/assoc>

- 関連するすべてのファイルタイプをします:

```bash
assoc
```

- 特定の拡張子に関連付けられているファイルの種類を表示します:

```bash
assoc .txt
```

- 特定の拡張子に関連付けられたファイルタイプを変更します:

```bash
assoc .txt=テキストファイル
```
