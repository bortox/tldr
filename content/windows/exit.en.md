> More information: <https://docs.microsoft.com/windows-server/administration/windows-commands/exit>.

- Quit the current CMD instance:

```bash
exit
```

- Quit the current batch script:

```bash
exit /b
```

- Quit using a specific exit code:

```bash
exit exit_code
```
