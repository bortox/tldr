> 更多信息：<https://docs.microsoft.com/windows-server/administration/windows-commands/color>.

- 将控制台颜色恢复至默认值：

```bash
color
```

- 列出可用的颜色值和详细信息：

```bash
color /?
```

- 将控制台前景色和背景色设置为指定颜色：

```bash
color 前景色值背景色值
```
