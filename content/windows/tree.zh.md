> 更多信息：<https://docs.microsoft.com/windows-server/administration/windows-commands/tree>.

- 显示当前目录的目录树：

```bash
tree
```

- 显示指定目录的目录树：

```bash
tree 目录的路径
```

- 显示目录中文件的目录树：

```bash
tree 目录的路径 /f
```

- 使用 ASCII 字符而不是扩展字符显示目录树：

```bash
tree 目录的路径 /a
```
