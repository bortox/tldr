> 更多資訊：<https://docs.microsoft.com/windows-server/administration/windows-commands/mkdir>.

- 建立一個資料夾：

```bash
mkdir 資料夾名稱
```

- 遞迴建立資料夾及子資料夾：

```bash
mkdir 資料夾/子資料夾名稱
```
