> 更多信息：<https://chocolatey.org/docs/commands-feature>.

- 显示可用的功能列表：

```bash
choco feature list
```

- 启用一个功能：

```bash
choco feature enable --name 功能名称
```

- 禁用一个功能：

```bash
choco feature disable --name 功能名称
```
