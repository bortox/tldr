> Informasi lebih lanjut: <https://ss64.com/osx/pbcopy.html>.

- Menempatkan konten file pada papan klip:

```bash
pbcopy < file
```

- Menempatkan hasil perintah pada papan klip:

```bash
find . -type t -name "*.png" | pbcopy
```
