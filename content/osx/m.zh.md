> 更多信息：<https://github.com/rgcr/m-cli>.

- 获取电池状态：

```bash
m battery status
```

- 关闭蓝牙：

```bash
m bluetooth off
```

- 列出可用于格式化的文件系统：

```bash
m disk filesystems
```

- 启用 Dock（桌面程序坞）的自动隐藏功能：

```bash
m dock autohide YES
```

- 禁用防火墙：

```bash
m firewall disable
```
