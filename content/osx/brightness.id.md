> Informasi lebih lanjut: <https://github.com/nriley/brightness>.

- Menampilkan brightness sekarang:

```bash
brightness -l
```

- Mengubah brightness menjadi 100%:

```bash
brightness 1
```

- Mengubah brightness menjadi 50%:

```bash
brightness 0.5
```
