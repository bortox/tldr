> 更多信息：<https://ss64.com/osx/md5.html>.

- 计算一个文件的 MD5 校验值：

```bash
md5 文件名
```

- 计算多个文件的 MD5 校验值：

```bash
md5 文件名 1 文件名 2
```

- 仅输出 MD5 校验值（无文件名）：

```bash
md5 -q 文件名
```

- 打印给定字符串的 MD5 校验值：

```bash
md5 -s 字符串
```
