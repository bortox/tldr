> More information: <https://www.manpagez.com/man/1/xed/>.

- Open file in Xcode:

```bash
xed file1
```

- Open file(s) in Xcode, create if it doesn't exist:

```bash
xed --create filename1
```

- Open a file in Xcode and jump to line number 75:

```bash
xed --line 75 filename
```
