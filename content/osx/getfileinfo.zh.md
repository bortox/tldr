> 更多信息：<https://www.unix.com/man-page/osx/1/GetFileInfo/>.

- 显示有关给定文件的信息：

```bash
GetFileInfo 路径/文件名
```

- 显示给定文件的创建日期和时间：

```bash
GetFileInfo -d 路径/文件名
```

- 显示给定文件的上次修改日期和时间：

```bash
GetFileInfo -m 路径/文件名
```

- 显示给定文件的创建者：

```bash
GetFileInfo -c 路径/文件名
```
