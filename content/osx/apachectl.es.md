> Más información: <https://www.unix.com/man-page/osx/8/apachectl/>.

- Inicia la tarea launchd `org.apache.httpd`:

```bash
apachectl start
```

- Finaliza la tarea launchd:

```bash
apachectl stop
```

- Finaliza e inicia la tarea launchd:

```bash
apachectl restart
```
