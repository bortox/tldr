> 更多信息：<https://github.com/nriley/brightness>.

- 显示当前亮度：

```bash
brightness -l
```

- 设置亮度到 100%:：

```bash
brightness 1
```

- 设置亮度到 50%:：

```bash
brightness 0.5
```
