- 列出所有可用的更新：

```bash
softwareupdate -l
```

- 下载并安装所有更新：

```bash
softwareupdate -ia
```

- 下载并安装所有推荐的更新：

```bash
softwareupdate -ir
```

- 下载并安装特定的应用程序：

```bash
softwareupdate -i 更新应用程序名
```
