> Mais informações: <https://www.manpagez.com/man/1/xed/>.

- Abre um ficheiro no Xcode:

```bash
xed ficheiro1
```

- Abre um ou mais ficheiros no Xcode, cria o ficheiro se o mesmo não existir:

```bash
xed -c ficheiro1
```

- Abre um ficheiro no Xcode e foca na linha 88:

```bash
xed -l 88 ficheiro
```
