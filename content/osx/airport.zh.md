> 更多信息：<https://ss64.com/osx/airport.html>.

- 显示当前的无线状态信息：

```bash
airport -I
```

- 在通道 1 上监察（嗅探）无线流量：

```bash
airport sniff 1
```

- 扫描可用的无线网络：

```bash
airport -s
```

- 与当前的 Airport 网络脱离连接：

```bash
sudo airport -z
```
