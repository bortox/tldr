> 更多信息：<https://github.com/joshfinnie/archey-osx>.

- 显示系统信息（彩色的）：

```bash
archey
```

- 显示系统信息（单色的）：

```bash
archey --nocolor
```

- 显示系统信息，使用 MacPorts（命令行软件安装管理工具 port）来替代 Homebrew（另一种更常用的 mac 命令行软件安装管理工具）：

```bash
archey --macports
```

- 显示系统信息，但不进行 IP 地址获取和验证：

```bash
archey --offline
```
