> More information: <https://github.com/rauchg/wifi-password>.

- Get the password for the Wi-Fi you are currently logged onto:

```bash
wifi-password
```

- Get the password for the Wi-Fi with a specific SSID:

```bash
wifi-password ssid
```

- Print only the password as output:

```bash
wifi-password -q
```
