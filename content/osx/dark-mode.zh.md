> 更多信息：<https://github.com/sindresorhus/dark-mode>.

- 切换深色模式（如果目前是关闭的，就把它打开，如果目前是打开的，就把它关闭）：

```bash
dark-mode
```

- 打开深色模式：

```bash
dark-mode on
```

- 关闭深色模式：

```bash
dark-mode off
```

- 检查深色模式是否开启：

```bash
dark-mode status
```
