> 另请参阅 `popd` 命令说明，以切换回原始目录。

- 切换到目录并将其添加到堆栈上：

```bash
pushd directory
```

- 切换堆栈上的第一个和第二个目录：

```bash
pushd
```

- 通过使第 5 个元素成为堆栈的顶部来旋转堆栈：

```bash
pushd +4
```
