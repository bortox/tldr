> 更多信息：<https://ss64.com/osx/sw_vers.html>.

- 打印所有可用信息（操作系统名称、版本号和构建版本）：

```bash
sw_vers
```

- 只打印操作系统的版本号：

```bash
sw_vers -productVersion
```

- 只打印构建版本的标识符：

```bash
sw_vers -buildVersion
```
