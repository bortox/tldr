> 更多信息：<http://webcam-tools.sourceforge.net>.

- 从网络摄像机拍照：

```bash
wacaw 文件名
```

- 录制视频：

```bash
wacaw --video 文件名 -D 录制多少秒
```

- 用自定义分辨率拍照：

```bash
wacaw -x 宽 -y 高 文件名
```

- 将刚拍摄的图像复制到剪贴板：

```bash
wacaw --to-clipboard
```

- 可用设备列表：

```bash
wacaw -L
```
