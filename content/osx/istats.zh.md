> 更多信息：<https://github.com/Chris911/iStats>.

- 显示所有的统计数据：

```bash
istats
```

- 显示所有 CPU 的统计数据：

```bash
istats cpu
```

- 显示所有风扇的统计数据：

```bash
istats fan
```

- 扫描和打印温度：

```bash
istats scan
```
