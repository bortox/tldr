> More information: <https://github.com/Chris911/iStats>.

- Show all the stats:

```bash
istats
```

- Show all CPU stats:

```bash
istats cpu
```

- Show all fan stats:

```bash
istats fan
```

- Scan and print temperatures:

```bash
istats scan
```
