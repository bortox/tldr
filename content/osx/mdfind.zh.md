> 更多信息：<https://ss64.com/osx/mdfind.html>.

- 按文件名查找文件：

```bash
mdfind -name 文件
```

- 按内容查找文件：

```bash
mdfind 查找的字符串
```

- 在给定目录中查找包含字符串的文件：

```bash
mdfind -onlyin 目录 字符串
```
