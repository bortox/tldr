> 更多信息：<https://ss64.com/osx/cal.html>.

- 打印本月日历：

```bash
cal
```

- 显示上个月，当前月，下个月的日历：

```bash
cal -3
```

- 显示指定月份的日历（月份为 1-12 月）：

```bash
cal -m 月
```

- 显示全年日历：

```bash
cal -y
```

- 显示指定某年的日历（年份为 4 个数字）：

```bash
cal 年
```

- 显示特定年和月的日历：

```bash
cal 月 年
```

- 显示指定年的复活节日期：

```bash
ncal -e 年
```
