> 注意：有关操作系统的其他信息，请尝试使用 `sw-vers` 命令。

> 更多信息：<https://ss64.com/osx/uname.html>.

- 打印硬件相关：架构信息和处理器：

```bash
uname -mp
```

- 打印软件相关信息：操作系统、版本号和版本：

```bash
uname -srv
```

- 打印系统的节点名称（主机名）：

```bash
uname -n
```

- 打印所有可用的系统信息（硬件、软件、节点名）：

```bash
uname -a
```
