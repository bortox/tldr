- 随机化文件中的行顺序并输出结果：

```bash
shuf 文件名
```

- 只输出结果的前 5 条：

```bash
shuf -n 5 文件名
```

- 将结果输出写入另一个文件：

```bash
shuf 文件名 -o 输出_文件名
```

- 生成范围（1-10）内的随机数：

```bash
shuf -i 1-10
```
