> 更多信息：<https://guides.cocoapods.org/terminal/commands.html>.

- 为当前项目初始化包含默认内容的 podfile：

```bash
pod init
```

- 下载并安装 pod 文件中定义的所有 pod（以前未安装）：

```bash
pod install
```

- 列出所有可用的 pod：

```bash
pod list
```

- 显示过时的 pod（当前安装的 pod）：

```bash
pod outdated
```

- 将当前安装的所有 pod 更新到其最新版本：

```bash
pod update
```

- 将特定（以前安装的）pod 更新为其最新版本：

```bash
pod update pod_名
```

- 从 Xcode 项目中删除 CocoaPods：

```bash
pod deintegrate xcode_项目
```
