> 更多信息：<https://www.manpagez.com/man/8/dmesg/>.

- 显示内核消息：

```bash
dmesg
```

- 显示此系统上有多少可用的物理内存：

```bash
dmesg | grep -i memory
```

- 一次显示一页内核消息：

```bash
dmesg | less
```
