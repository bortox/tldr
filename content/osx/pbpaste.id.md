> Informasi lebih lanjut: <https://ss64.com/osx/pbpaste.html>.

- Menulis konten papan klip ke dalam sebuah file:

```bash
pbpaste > file
```

- Menggunakan konten papan klip sebagai input bagi sebuah perintah:

```bash
pbpaste | grep foo
```
