> Maggiori informazioni: <https://www.unix.com/man-page/osx/8/apachectl/>.

- Avvia il demone `org.apache.httpd`:

```bash
apachectl start
```

- Ferma il demone:

```bash
apachectl stop
```

- Riavvia il demone:

```bash
apachectl restart
```
