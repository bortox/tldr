> 更多信息：<https://github.com/rauchg/wifi-password>.

- 获取你当前登录的 wifi 的密码：

```bash
wifi-password
```

- 获取特定 SSID 的 wifi 的密码：

```bash
wifi-password ssid
```

- 仅输出密码：

```bash
wifi-password -q
```
