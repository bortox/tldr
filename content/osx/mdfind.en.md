> More information: <https://ss64.com/osx/mdfind.html>.

- Find a file by its name:

```bash
mdfind -name file
```

- Find a file by its content:

```bash
mdfind "query"
```

- Find a file containing a string, in a given directory:

```bash
mdfind -onlyin directory "query"
```
