> 更多信息：<https://www.manpagez.com/man/1/xed/>.

- 用 Xcode 打开文件：

```bash
xed 文件名
```

- 在 Xcode 中打开文件，如果不存在则创建：

```bash
xed -c 文件名
```

- 在 Xcode 中打开一个文件并跳转到第 75 行：

```bash
xed -l 75 文件名
```
