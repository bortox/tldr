> 更多信息：<https://www.unix.com/man-page/osx/8/apachectl/>.

- 启动 org.apache.httpd 服务：

```bash
apachectl start
```

- 停止已启动的服务：

```bash
apachectl stop
```

- 重新启动服务：

```bash
apachectl restart
```
